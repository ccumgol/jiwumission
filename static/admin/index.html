🔑 GitHub 토큰이 없으신가요? 생성 방법
  1. GitHub Token Settings로 이동합니다.
  2. Token name을 입력하고, Repository access에서 Only select repositories를 누른 후 jiwumission을 고릅니다.
  3. Permissions에서 Repository permissions -> Contents 권한을 Read and Write로 부여합니다.
  4. 하단의 Generate token 버튼을 누르고 생성된 토큰을 복사하여 위에 붙여넣습니다.

설정 파일을 불러오는 중입니다...

저장 및 반영 방식

각 설정 탭의 저장 버튼을 누르면 GitHub 저장소의 설정 파일이 직접 커밋되어 실시간 반영됩니다. 반영 완료(빌드 성공)까지 약 1~2분이 소요됩니다.
