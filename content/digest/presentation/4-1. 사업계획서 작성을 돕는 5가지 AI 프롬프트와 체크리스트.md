## 사업계획서 작성을 돕는 5가지 AI 프롬프트

AI를 활용하면 사업계획서 작성 시간을 70% 단축할 수 있습니다. 다음은 실전에서 바로 사용할 수 있는 프롬프트입니다.

### 프롬프트 1: 시장 분석 보고서 생성

`당신은 시장 조사 전문가입니다. 다음 사업에 대한 상세 시장 분석 보고서를 작성해주세요.

사업 정보:
- 업종: [카페]
- 위치: [서울 강남구 역삼동]
- 목표 고객: [30~40대 직장인]
- 특징: [조용한 작업 공간 + 도서 대여]

포함할 내용:
1. 산업 전체 규모 및 성장률
2. 목표 시장 세분화 (4~5개 고객군)
3. 각 세그먼트별 크기, 특징, 소비 패턴
4. 주요 경쟁사 3~5개 분석 (강점/약점)
5. 시장 진입 장벽
6. 향후 3년 트렌드 전망

출력 형식: 표, 그래프 설명, 구체적 수치 포함`

**실제 사용 예:**
"당신은 시장 조사 전문가입니다. 서울 강남구 역삼동에 조용한 작업 공간과 도서 대여를 제공하는 카페를 오픈하려 합니다. 30~40대 직장인을 주요 타겟으로 합니다. 상세 시장 분석 보고서를 작성해주세요..."

### 프롬프트 2: 재무 모델 시뮬레이션

`당신은 재무 분석 전문가입니다. 다음 사업의 3년 재무 모델을 생성해주세요.

초기 투자:
- 인테리어: $83,000
- 장비: $40,000
- 기타: $44,000

월간 수익:
- 커피/음료: $17,000
- 베이커리: $6,700
- 기타: $3,300

월간 비용:
- 재료비: 매출의 30%
- 인건비: $9,300
- 공과금: $1,700
- 기타: $1,700

출력할 내용:
1. 월별 손익계산서 (36개월)
2. 현금흐름표
3. 손익분기점 분석
4. 시나리오 분석 (낙관/기본/보수/비관)
5. 주요 재무 비율 (ROI, IRR, 순이익률)

출력 형식: 표 형식, 월별 추이 그래프`

### 프롬프트 3: 경쟁 우위 분석

`당신은 전략 컨설턴트입니다. 다음 사업의 경쟁 우위를 분석하고 포지셔닝 전략을 제안해주세요.

우리 사업:
- [그레이스 북카페]
- 특징: [3,300 sq ft 넓은 공간, 조용함, 무료 주차, 도서 대여]
- 가격: [아메리카노 $3.50]

경쟁사:
1. 스타벅스: 브랜드 강력, 시끄러움, 주차 불가, $5.50
2. 투썸플레이스: 디저트 좋음, 공간 좁음, 주차 불가, $5.00
3. 로컬 카페: 아늑함, 낡은 시설, 주차 없음, $4.50

분석할 내용:
1. 경쟁사 대비 우리의 강점/약점
2. 차별화 요소 3가지
3. 포지셔닝 맵 (가격 vs 품질, 공간 vs 서비스 등)
4. 경쟁 전략 (차별화/원가우위/집중화)
5. 지속 가능한 경쟁 우위 확보 방안

출력 형식: 비교표, 포지셔닝 맵, 전략 설명`

### 프롬프트 4: 위험 관리 매트릭스

`당신은 리스크 관리 전문가입니다. 다음 사업의 위험 요소를 분석하고 대응 방안을 제시해주세요.

사업 정보:
- 업종: [카페]
- 위치: [교회 건물 지하]
- 초기 투자: $167,000
- 주요 차별화: [조용한 환경, 도서 대여]

위험 카테고리:
1. 시장 위험 (경쟁, 수요 변화)
2. 운영 위험 (인력, 품질, 공급)
3. 재무 위험 (현금흐름, 비용 초과)
4. 평판 위험 (리뷰, 민원)
5. 외부 위험 (규제, 경제)

각 위험별 분석:
- 발생 확률 (높음/중간/낮음)
- 영향도 (높음/중간/낮음)
- 구체적 시나리오
- 예방 조치
- 대응 방안
- 비상 계획

출력 형식: 위험 매트릭스 표, 우선순위 목록, 대응 계획서`

### 프롬프트 5: Executive Summary 작성

`당신은 사업계획서 작성 전문가입니다. 다음 정보를 바탕으로 투자자용 Executive Summary 2페이지를 작성해주세요.

사업 정보:
- 사업명: [그레이스 북카페]
- 컨셉: [조용한 작업 공간 + 도서 대여]
- 위치: [서울 강남구 역삼동, 3,300 sq ft]
- 목표 고객: [30~40대 직장인 8,000명]

시장:
- TAM: $670M, SAM: $27M, SOM: $800K
- 경쟁사: 스타벅스, 투썸, 로컬 카페

경쟁 우위:
- 넓은 공간, 조용함, 무료 주차 30대, 임대료 무료

재무:
- 초기 투자: $167,000
- 1년차 매출: $324,000
- 손익분기: 8개월
- 3년 누적 순이익: $140,000
- IRR: 52%

팀:
- 김영희 (카페 운영 5년, 바리스타 1급)
- 박철수 (CPA, 재무)

작성 원칙:
1. 첫 문단에 핵심 메시지 집중
2. 구체적 숫자 사용
3. 투자자 관점 (왜 성공? 얼마 벌? 누가?)
4. 2페이지 이내, 읽기 쉽게

출력 형식: 완성된 Executive Summary 텍스트`

## 제출 전 최종 체크리스트

사업계획서를 제출하기 전, 다음 항목들을 반드시 확인하십시오.

### 내용 완성도 체크 (15개 항목)

**□ 시장 분석**

- [ ] 목표 시장 규모가 구체적 수치로 제시되어 있는가? (TAM/SAM/SOM)
- [ ] 고객 세그먼트가 명확히 구분되어 있는가? (최소 3개)
- [ ] 경쟁사 분석이 3개 이상 포함되어 있는가?
- [ ] 우리의 경쟁 우위가 명확히 설명되어 있는가?

**□ 재무 계획**

- [ ] 초기 투자 내역이 항목별로 세분화되어 있는가?
- [ ] 월별 손익 계획이 최소 12개월 이상 제시되어 있는가?
- [ ] 손익분기점이 명확히 계산되어 있는가?
- [ ] 시나리오 분석이 포함되어 있는가? (낙관/기본/보수)
- [ ] ROI, IRR 등 주요 재무 지표가 계산되어 있는가?

**□ 실행 계획**

- [ ] 팀원의 구체적 경력과 자격이 명시되어 있는가?
- [ ] 월별 실행 일정이 구체적으로 제시되어 있는가?
- [ ] 주요 위험 요소와 대응책이 포함되어 있는가?

**□ 전체 구성**

- [ ] Executive Summary가 2~3페이지 이내인가?
- [ ] 전체 분량이 25~40페이지 범위인가?
- [ ] 모든 주장이 데이터나 근거로 뒷받침되는가?

### 형식 및 가독성 체크 (10개 항목)

**□ 문서 형식**

- [ ] 표지에 사업명, 작성자, 날짜가 명확히 표시되어 있는가?
- [ ] 목차가 페이지 번호와 함께 정확히 작성되어 있는가?
- [ ] 모든 표와 그래프에 제목과 출처가 표시되어 있는가?
- [ ] 폰트가 일관적이고 읽기 쉬운가? (본문 11~12pt)

**□ 데이터 정확성**

- [ ] 모든 금액 단위가 일관적인가? ($로 통일)
- [ ] 모든 거리 단위가 일관적인가? (miles로 통일)
- [ ] 모든 면적 단위가 일관적인가? (sq ft로 통일)
- [ ] 계산 오류가 없는가? (합계, 비율 등)

**□ 최종 검토**

- [ ] 오타나 문법 오류가 없는가?
- [ ] PDF로 변환 시 레이아웃이 깨지지 않는가?

### 투자자 관점 체크 (5개 항목)

**□ 핵심 질문에 답했는가?**

- [ ] "왜 이 사업이 성공할 것인가?" - 시장 기회와 경쟁 우위로 답변
- [ ] "얼마나 벌 것인가?" - 구체적 재무 전망으로 답변
- [ ] "이 팀이 할 수 있는가?" - 검증 가능한 역량으로 답변
- [ ] "최악의 경우는?" - 위험 관리와 출구 전략으로 답변
- [ ] "내 돈을 언제 돌려받는가?" - 회수 일정과 방법으로 답변
