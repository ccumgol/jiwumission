# Google Gemini Gems 상세 활용 매뉴얼

Gemini Gems는 특정 목적에 맞춰 제미나이의 역할과 동작 방식을 사전에 설정해 두는 **'맞춤형 AI 비서'** 기능입니다. 매번 긴 프롬프트를 입력할 필요 없이, 한 번 설정해 둔 Gem을 호출하여 일관된 고품질의 답변을 얻을 수 있습니다.

## 1. 버전별 이용 권한 (무료 vs 유료)

Gems 기능은 기본적으로 **Gemini Advanced(유료)** 사용자를 위해 설계되었으나, 최근 정책에 따라 무료 사용자도 일부 기능을 이용할 수 있게 되었습니다.

| 구분 | Gemini (무료 버전) | Gemini Advanced (유료 버전) |
| --- | --- | --- |
| **Gems 사용** | 구글에서 기본 제공하는 Gems 사용 가능 | 모든 커스텀 Gems 사용 가능 |
| **커스텀 Gems 생성** | 제한적 또는 불가 (구독 권장) | **제한 없이 생성 및 공유 가능** |
| **지식 업로드 (Knowledge)** | 지원하지 않음 | **PDF, CSV, Docx 등 파일 기반 학습 지원** |
| **최신 모델 적용** | 기본 모델 적용 | **Gemini 1.5 Pro 등 최상위 모델 기반 작동** |

## 2. Gems의 역할 및 주요 기능

### 2.1 주요 역할

- **반복 작업 효율화:** 자주 사용하는 프롬프트를 고정하여 업무 시간을 단축합니다.
  - *예시: 매일 아침 수신된 영문 뉴스레터를 한국어로 번역하고 핵심 요약 3줄을 추출하는 작업을 자동화.*
- **전문성 강화:** 특정 분야(코딩, 작문, 마케팅 등)의 전문가 페르소나를 부여해 답변의 깊이를 더합니다.
  - *예시: '10년 차 법률 자문가' 페르소나를 설정하여 계약서의 독소 조항이나 법적 리스크를 검토.*
- **일관성 유지:** 동일한 톤앤매너와 출력 형식을 유지하여 협업 및 문서화에 유리합니다.
  - *예시: 모든 고객 응대 메일 초안을 '친절하고 정중한 격식체'와 '문제 해결 중심'의 구조로 통일하여 작성.*

### 2.2 주요 기능

- **커스텀 지침 (Custom Instructions):** AI가 누구인지, 무엇을 해야 하는지, 어떻게 답해야 하는지 상세히 정의합니다.
- **지식 업로드 (Knowledge):** PDF, Docx, CSV 등 파일을 업로드하여 해당 데이터를 기반으로 답변하게 할 수 있습니다. (Gemini Advanced 전용)
- **도구 연동:** Google Workspace(Gmail, Drive 등)와 연동하여 실제 데이터를 참조하거나 이메일을 초안 작성을 돕습니다.

## 3. Gems 설정 및 활용 프로세스

1. **접속:** [gemini.google.com](https://gemini.google.com/)에 접속합니다.
2. **Gems 관리:** 왼쪽 사이드바에서 **'Gems 탐색하기'** 또는 **'새 Gem'** 버튼을 클릭합니다.
3. **기본 정보 입력:** Gem의 이름과 해당 Gem이 수행할 아이콘/색상을 선택합니다.
4. **지침 작성:** '요청 사항(Instructions)' 박스에 AI의 동작 규칙을 입력합니다. (4항의 가이드 참고)
5. **미리보기 및 테스트:** 오른쪽 창에서 실제 대화를 나누며 의도대로 작동하는지 확인합니다.
6. **저장:** 만족스러운 결과가 나오면 **'저장'**을 클릭하여 나만의 Gem을 생성합니다.

## 4. 효과적인 프롬프트(지침) 작성법: 4단계 프레임워크

전문가들이 사용하는 **Persona + Task + Context + Format** 법칙을 적용하면 훨씬 정교한 Gem을 만들 수 있습니다.

| 요소 | 설명 | 예시 |
| --- | --- | --- |
| **Persona (페르소나)** | AI의 역할과 성격 정의 | "당신은 10년 차 IT 기술 블로그 전문 에디터입니다." |
| **Task (작업)** | 구체적으로 수행할 목표 | "제공된 기술 문서를 대중이 이해하기 쉬운 블로그 포스트로 요약하세요." |
| **Context (맥락)** | 제약 사항 및 배경 정보 | "독자는 비전공자입니다. 전문 용어는 반드시 각주를 달아 설명하세요." |
| **Format (출력 형식)** | 결과물의 구조와 스타일 | "제목, 서론, 본문(3개 섹션), 결론의 구조로 작성하고 마크다운을 사용하세요." |

## 5. 복잡한 지침 작성 예시 (고급 사례)

### 예시 1: '전략적 마케팅 콘텐츠 기획자' Gem

> **[지침 내용]**
>
> - **역할:** 당신은 고도의 전략적 사고를 가진 디지털 마케팅 팀장입니다.
> - **수행 프로세스:**
>   1. 사용자가 주제를 제시하면 타겟 페르소나 3종을 분석합니다.
>   2. 각 페르소나별 SEO 최적화 키워드를 5개씩 추출합니다.
>   3. 최종적으로 인스타그램, 블로그, 뉴스레터용 맞춤형 카피라이팅을 생성합니다.
> - **금지 사항:** 추상적인 표현은 피하고 데이터를 강조하세요.

## 6. 유지보수 및 팁

- **반복 수정:** 의도와 다른 답을 한다면 "이러한 질문에는 ~하게 대답하지 마세요"라는 예외 규칙을 추가하세요.
- **제미나이로 다시 쓰기:** 지침 입력 창 하단의 '제미나이를 사용해 다시 작성하기' 버튼으로 초안을 보정하세요.
