## Chapter 1. Notion 시작하기

**학습 목표:** Notion이 무엇인지 이해하고, 계정을 만들어 화면을 자유롭게 둘러볼 수 있다.

**다룰 내용**

- Notion이란 무엇인가 (문서·메모·데이터베이스·협업을 한 곳에서)
- 계정 만들기와 요금제 개념 (개인/팀)
- 화면 구성 살펴보기: 사이드바, 상단바, 본문 영역
- 워크스페이스 · 팀스페이스 · 페이지의 관계
- 데스크톱 앱 / 웹 / 모바일 차이

**용어정리(예정):** 워크스페이스, 팀스페이스, 사이드바, 페이지

**FAQ(예정):** 무료로 어디까지 쓸 수 있나요? / 회사 계정과 개인 계정을 나눠야 하나요? / 앱과 웹 중 뭘 써야 하나요?

**Tips(예정):** 단축키로 빠르게 검색하기, 다크모드 설정, 즐겨찾기 활용

## Chapter 2. 페이지와 블록

**학습 목표:** 노션의 모든 것이 '블록'으로 이루어졌음을 이해하고, 페이지를 만들고 정리할 수 있다.

**다룰 내용**

- '블록' 개념 — 노션의 최소 작업 단위
- 페이지 만들기 · 하위 페이지(중첩) 개념
- 블록 추가·이동·삭제 (드래그, `/` 슬래시 명령어)
- 블록 유형 훑어보기 (텍스트, 제목, 목록, 이미지 등)
- 페이지 아이콘·커버·이모지로 꾸미기

**용어정리(예정):** 블록, 슬래시 명령어(/), 하위 페이지, 커버

**FAQ(예정):** 블록이 안 옮겨져요 / 페이지를 실수로 지웠어요(휴지통) / 페이지와 블록의 차이가 뭔가요?

**Tips(예정):** `/`로 모든 걸 호출하기, 여러 블록 한 번에 선택·이동, 블록 색상 지정

---

## Chapter 3. 텍스트 작성과 서식

**학습 목표:** 읽기 좋은 문서를 만들 수 있도록 서식과 정리 도구를 활용할 수 있다.

**다룰 내용**

- 텍스트 서식: 굵게·기울임·색상·강조
- 제목(Heading)과 구분선으로 구조 만들기
- 목록: 글머리·번호·체크박스(할 일)
- 토글, 콜아웃, 인용으로 정보 접기·강조하기
- 표(단순 표)와 열 나누기(Columns) 레이아웃
- 이미지·파일·링크·북마크 삽입

**용어정리(예정):** 토글, 콜아웃, 체크박스, 열(Column) 레이아웃

**FAQ(예정):** 마크다운 문법을 몰라도 되나요? / 표와 데이터베이스는 뭐가 다른가요? / 붙여넣기하면 서식이 깨져요

**Tips(예정):** 마크다운 단축 입력(`#`, `-`, `[]`), 토글로 긴 문서 접기, 콜아웃으로 주의사항 강조

## Chapter 4. 워크스페이스 정리와 협업

**학습 목표:** 페이지를 체계적으로 정리하고, 다른 사람과 함께 작업할 수 있다.

**다룰 내용**

- 사이드바로 페이지 구조 잡기 (중첩·정렬)
- 페이지 이동·복제·즐겨찾기
- 공유와 권한 (보기·댓글·편집·전체 허용)
- 댓글·멘션(@)으로 소통하기
- 링크 공유 vs 초대, 게스트 개념
- 버전 기록으로 되돌리기

**용어정리(예정):** 권한, 멘션(@), 게스트, 버전 기록

**FAQ(예정):** 공유 링크를 아무나 볼 수 있나요? / 실수한 편집을 되돌릴 수 있나요? / 특정 사람만 편집하게 하려면?

**Tips(예정):** 팀 온보딩용 홈 페이지 만들기, 권한 최소화 원칙, 멘션으로 담당자 지정

---

## Chapter 5. 데이터베이스 기초

**학습 목표:** 데이터베이스가 일반 표와 어떻게 다른지 이해하고, 첫 데이터베이스를 만들 수 있다.

**다룰 내용**

- 데이터베이스란? (표처럼 보이지만 각 행이 하나의 페이지)
- 인라인 vs 전체 페이지 데이터베이스
- 행(항목) 추가와 각 행 열어보기
- 정렬·필터·검색으로 원하는 데이터 찾기
- 실무 예시: 할 일 목록, 연락처, 자료 관리

**용어정리(예정):** 데이터베이스, 행/항목, 인라인, 필터, 정렬

**FAQ(예정):** 엑셀과 뭐가 다른가요? / 행 하나가 왜 페이지가 되나요? / 데이터가 많아지면 느려지지 않나요?

**Tips(예정):** 템플릿으로 반복 항목 만들기, 필터 저장, 그룹화로 한눈에 보기

---

## Chapter 6. 데이터베이스 뷰

**학습 목표:** 하나의 데이터를 목적에 맞는 여러 형태로 볼 수 있다.

**다룰 내용**

- 뷰(View) 개념 — 같은 데이터, 다른 표현
- 주요 뷰: 표, 보드(칸반), 캘린더, 리스트, 갤러리, 타임라인
- 새 뷰 추가하고 뷰별 필터·정렬 설정
- 대시보드 뷰로 요약 보기 (최신 버전 기능)
- 언제 어떤 뷰를 쓰면 좋은가

**용어정리(예정):** 뷰, 보드(칸반), 갤러리, 타임라인, 대시보드

**FAQ(예정):** 뷰를 바꾸면 데이터가 사라지나요? / 사람마다 다른 뷰를 볼 수 있나요? / 보드는 언제 쓰나요?

**Tips(예정):** 진행 단계 관리는 보드 뷰, 일정 관리는 캘린더/타임라인, 뷰마다 필터 다르게

## Chapter 7. 속성(Property) 다루기

**학습 목표:** 데이터의 성격에 맞는 속성을 골라 구조화된 정보를 만들 수 있다.

**다룰 내용**

- 속성이란? (열 = 데이터의 종류)
- 기본 속성: 텍스트, 숫자, 선택/다중선택, 날짜, 체크박스, 사람, 파일
- 상태(Status) 속성으로 진행 관리
- 자동 속성: 생성일시, 최종 편집일시, 생성자
- 속성 추가·숨기기·순서 바꾸기

**용어정리(예정):** 속성, 선택/다중선택, 상태(Status), 자동 속성

**FAQ(예정):** 선택과 다중선택은 어떻게 다른가요? / 속성을 지우면 데이터도 지워지나요? / 상태와 선택 중 뭘 쓰죠?

**Tips(예정):** 색상으로 상태 구분, 자주 안 쓰는 속성 숨기기, 다중선택으로 태그 관리

## Chapter 8. 관계형(Relation)과 롤업(Rollup)

**학습 목표:** 두 데이터베이스를 연결하고, 연결된 데이터를 불러와 집계할 수 있다.

**다룰 내용**

- 관계형(Relation) 개념 — 데이터베이스끼리 연결하기
- 예시: '프로젝트' ↔ '할 일', '고객' ↔ '주문'
- 양방향 관계 설정
- 롤업(Rollup)으로 연결된 값 가져오기·합계·개수 세기
- 실무 활용 시나리오

**용어정리(예정):** 관계형(Relation), 롤업(Rollup), 양방향 관계, 집계

**FAQ(예정):** 관계형과 그냥 링크는 뭐가 다른가요? / 롤업이 빈 값으로 나와요 / 관계를 잘못 연결하면?

**Tips(예정):** 관계형으로 중복 입력 없애기, 롤업으로 진행률 계산, 필터와 함께 쓰기

---

## Chapter 9. 수식(Formula) 입문

**학습 목표:** 간단한 수식으로 값을 자동 계산하고, 자연어로 수식을 만들 수 있다.

**다룰 내용**

- 수식 속성이란? (다른 속성을 계산해 새 값 만들기)
- 기본 개념: 속성 참조, 숫자·텍스트·날짜·불리언(참/거짓)
- 자주 쓰는 함수 맛보기: `if`, `dateBetween`, `format`, `prop`
- 자연어로 수식 작성하기 (최신 기능 — 설명하면 Notion이 수식 생성)
- 실습 예시: D-day, 진행률, 조건부 표시

**용어정리(예정):** 수식(Formula), 함수, 불리언, 속성 참조(prop)

**FAQ(예정):** 프로그래밍을 몰라도 되나요? / 수식 오류는 어떻게 읽나요? / 롤업 값도 수식에 쓸 수 있나요?

**Tips(예정):** 자연어로 먼저 만들고 다듬기, 자주 쓰는 수식 저장, 단계별로 나눠 검증

## Chapter 10. 자동화와 AI 활용

**학습 목표:** 반복 작업을 자동화하고, Notion AI로 업무 속도를 높일 수 있다.

**다룰 내용**

- 버튼(Button)으로 반복 작업 한 번에 실행
- 데이터베이스 자동화 (특정 조건이 되면 자동 실행)
- Notion AI 기본: 요약·번역·초안 작성
- AI 에이전트 개념 소개 (최신 기능 — 지시를 주면 작업 수행)
- 템플릿과 자동화로 나만의 업무 시스템 완성하기

**용어정리(예정):** 버튼, 자동화(Automation), Notion AI, 에이전트

**FAQ(예정):** AI 기능은 유료인가요? / 자동화가 실행이 안 돼요 / 버튼과 템플릿은 뭐가 다른가요?

**Tips(예정):** 자주 쓰는 절차를 버튼으로, 회의록 요약은 AI에게, 작게 시작해 점차 자동화
