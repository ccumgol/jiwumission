**📑 한눈에 보는 주요 파일 확장자 가이드**

파일 이름 뒤에 붙는 점(.)과 세네 글자의 알파벳을 **확장자**라고 합니다. 확장자는 컴퓨터에게 "이 파일은 이런 종류니까 이 프로그램으로 열어줘!"라고 알려주는 이름표와 같습니다.

**1. 문서 파일 (Documents)**

가장 많이 접하게 되는 업무 및 학습용 파일들입니다.

| **확장자** | **설명** | **주요 실행 프로그램** |
| --- | --- | --- |
| **.docx** | 워드 문서 | 마이크로소프트 워드 (Word) |
| **.xlsx** | 엑셀 스프레드시트 | 마이크로소프트 엑셀 (Excel) |
| **.pptx** | 파워포인트 슬라이드 | 마이크로소프트 파워포인트 (PPT) |
| **.pdf** | 전자 문서 표준 | 어도비 애크로뱃, 웹 브라우저 |
| **.hwp / .hwpx** | 한글 문서 | 한글과컴퓨터 한글 |
| **.txt** | 일반 텍스트 | 메모장(Win), 텍스트 편집기(Mac) |

**2. 이미지 파일 (Images)**

사진이나 그래픽 요소들이 포함된 파일입니다.

| **확장자** | **설명** | **특징** |
| --- | --- | --- |
| **.jpg / .jpeg** | 일반 사진 | 용량이 작아 웹이나 모바일에서 가장 많이 쓰임 |
| **.webp** | 고화질/투명 이미지 | 배경을 투명하게 만들 수 있어 로고 등에 쓰임 |
| **.gif** | 움직이는 이미지 | 흔히 말하는 '움짤' 파일 |
| **.svg** | 벡터 그래픽 | 확대해도 깨지지 않는 로고나 아이콘용 |

**3. 오디오 및 영상 파일 (Media)**

음악을 듣거나 동영상을 볼 때 사용하는 파일입니다.

| **확장자** | **설명** | **특징** |
| --- | --- | --- |
| **.mp4** | 표준 동영상 | 대부분의 기기에서 재생되는 가장 대중적인 영상 포맷 |
| **.mov** | 애플 영상 포맷 | 주로 맥(Mac)이나 아이폰 환경에서 많이 쓰임 |
| **.mp3** | 표준 음악 파일 | 고음질을 유지하면서 용량을 줄인 음악 파일 |
| **.wav** | 무압축 오디오 | 용량은 크지만 음질 손실이 없는 원음 파일 |

**4. 압축 파일 (Compressed)**

여러 개의 파일을 하나로 묶어 용량을 줄인 파일입니다.

| **확장자** | **설명** | **사용법** |
| --- | --- | --- |
| **.zip** | 표준 압축 포맷 | 윈도우와 맥 모두 별도 프로그램 없이 풀기 가능 |
| **.7z / .rar** | 고효율 압축 포맷 | 별도의 압축 프로그램(반디집, 알집 등)이 필요함 |

**5. 실행 및 설치 파일 (System/Apps)**

프로그램을 설치하거나 실행할 때 사용하는 파일입니다. **(주의가 필요함)**

| **확장자** | **설명** | **운영체제** |
| --- | --- | --- |
| **.exe** | 프로그램 실행 파일 | **Windows 전용** |
| **.msi** | 프로그램 설치 패키지 | **Windows 전용** |
| **.dmg** | 디스크 이미지 (설치) | **macOS 전용** |
| **.app** | 응용 프로그램 | **macOS 전용** |

**💡 초보자를 위한 확장자 주의사항**

- **함부로 바꾸지 마세요:** 파일 이름 뒤의 확장자를 강제로 바꾸면(예: .jpg를 .docx로) 파일이 손상되어 열리지 않을 수 있습니다.
- **모르는 .exe 파일은 조심하세요:** 모르는 사람이 보낸 메일에 실행 파일(.exe)이 있다면 바이러스일 확률이 높으니 절대 열지 마세요.
- **확장자가 안 보인다면?**
  - **Windows:** 탐색기 상단 [보기] > [파일 확장명] 체크
  - **macOS:** Finder 설정 > [고급] > [모든 파일 확장자 보기] 체크
