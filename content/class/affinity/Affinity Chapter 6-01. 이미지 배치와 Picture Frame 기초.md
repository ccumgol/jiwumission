## 이 파트에서 배울 내용

- 교회 주보에서 이미지와 도형이 맡는 역할을 이해합니다.
- `File → Place` 명령으로 이미지를 배치합니다.
- Picture Frame을 만들고 이미지의 위치·크기·자르기를 조절합니다.
- 이미지 비율, 회전, 반전 같은 기본 조작을 익힙니다.

---

## 1. 그래픽 요소의 역할

교회 주보에서 텍스트만으로는 전달하기 어려운 내용이 있습니다. 이미지와 도형은 다음 역할을 합니다.

| 역할 | 예시 |
| --- | --- |
| **시각적 계층 강조** | 배경 색상 띠로 섹션을 나눔 |
| **정보 전달 보완** | 예배당 위치 안내 지도, QR 코드 |
| **브랜드 아이덴티티 구현** | 교회 로고, 색상 테마 |
| **감성적 연결** | 예배 분위기의 사진, 계절 이미지 |
| **구분선 및 장식** | 가는 수평선, 모서리 장식 |

기본 워크플로우는 다음과 같습니다.

`[이미지/그래픽 파일] → Place로 배치 → Picture Frame으로 제어
        ↓
[도형] → Shape Tool로 그리기 → Fill/Stroke로 색상 설정
        ↓
[레이어 패널] → 순서·그룹·잠금 관리
        ↓
[Align 패널] → 정렬·배분으로 레이아웃 완성`

## 2. 이미지 배치하기 — Place 명령

### 2.1 지원 이미지 파일 형식

| 형식 | 특징 | 인쇄 적합성 |
| --- | --- | --- |
| **TIFF (.tif)** | 무손실, 고품질, 레이어 지원 | ✅ 최우선 권장 |
| **PDF (.pdf)** | 벡터+래스터 혼합 가능 | ✅ 우수 |
| **PSD (.psd)** | Photoshop 레이어 파일 | ✅ 우수 |
| **PNG (.png)** | 무손실, 투명 배경 지원 | ✅ 적합 (300DPI 이상) |
| **JPEG (.jpg)** | 손실 압축, 파일 경량 | ⚠️ 고화질 원본만 |
| **SVG (.svg)** | 벡터, 무한 확대 가능 | ✅ 로고·아이콘에 최적 |
| **EPS (.eps)** | 벡터, 레거시 형식 | ✅ 적합 |

> ⚠️ **JPEG 주의사항:** JPEG는 저장할 때마다 품질이 저하됩니다. 교회 로고나 중요 이미지는 PNG 또는 TIFF로 보관하세요.

### 2.2 Place 명령으로 이미지 배치

**경로**

- `File` → `Place`
- 단축키: `⌘ Cmd + D` / `Ctrl + D`

**단계**

1. `File` → `Place` 선택
2. 파일 탐색 창에서 이미지 파일 선택 → `Open`
3. 커서가 이미지 불러오기 모드로 변경됨
4. 다음 중 하나 선택
  - **클릭:** 이미지 원본 크기로 배치
  - **드래그:** 드래그한 영역 크기에 맞게 배치
  - **기존 Picture Frame 클릭:** 해당 프레임 안에 이미지 배치

### 2.3 여러 이미지 한 번에 배치

1. `File` → `Place`
2. 파일 탐색 창에서 여러 파일 선택
  - Mac: `⌘ Cmd + 클릭`
  - Windows: `Ctrl + 클릭`
3. `Open`
4. 클릭할 때마다 하나씩 배치

> 💡 Finder 또는 파일 탐색기에서 이미지 파일을 Affinity 문서로 직접 드래그 앤 드롭할 수도 있습니다.

### 2.4 Linked와 Embedded

| 방식 | 설명 | 파일 크기 | 공유 시 |
| --- | --- | --- | --- |
| **Linked (링크)** | 이미지 파일이 외부에 있고 Affinity가 참조 | 작음 | 이미지 파일도 함께 전달 필요 |
| **Embedded (임베드)** | 이미지가 Affinity 파일 내부에 복사됨 | 커짐 | 파일 하나로 자체 완결 |

**권장 방식**

- **작업 중:** Linked
- **인쇄소 제출 전:** Embedded 또는 Package 기능으로 이미지 폴더 함께 묶기

## 3. Picture Frame Tool

Picture Frame은 이미지가 들어갈 공간을 미리 예약하는 사각형 또는 타원형 박스입니다. 이미지를 직접 배치하는 것보다 더 정교하게 제어할 수 있습니다.

### 3.1 Picture Frame 종류

| 도구 이름 | 설명 |
| --- | --- |
| **Picture Frame Rectangle** | 사각형 이미지 프레임 |
| **Picture Frame Ellipse** | 원형/타원형 이미지 프레임 |

> 💡 사각형과 타원형 외의 모양으로 이미지 프레임을 만들려면 도형을 먼저 그린 뒤 `Layer → New Picture Frame from Shape`를 사용합니다.

### 3.2 Picture Frame 생성

1. 도구 모음에서 **Picture Frame Rectangle** 선택
2. 문서에서 드래그하여 프레임 생성
3. 빗금이 채워진 박스로 표시되면 이미지 대기 상태입니다.

### 3.3 Picture Frame에 이미지 배치

**방법 1 — Place 사용**

1. `File → Place` 실행
2. 이미지 선택
3. Picture Frame 위 클릭

**방법 2 — 드래그 앤 드롭**

- 파일 탐색기에서 이미지를 Picture Frame 위로 드래그합니다.

### 3.4 프레임과 이미지 각각 제어하기

| 목적 | 방법 |
| --- | --- |
| **프레임 크기/위치 조절** | Move Tool(`V`)로 프레임 클릭 → 핸들 드래그 |
| **이미지 위치 조절** | 프레임 더블클릭 → 이미지 이동 모드 → 드래그 |
| **이미지 크기 조절** | 프레임 더블클릭 → 이미지 선택 → 핸들 드래그 |
| **이미지 편집 모드 종료** | `Escape` 키 |

`[프레임 선택 상태] → 파란 테두리 → 프레임 이동/크기 조절
       ↓ 더블클릭
[이미지 편집 모드] → 주황 테두리 → 이미지 위치/크기 조절
       ↓ Escape
[프레임 선택 상태]`

### 3.5 이미지 채우기 방식

| 옵션 | 설명 |
| --- | --- |
| **Fit (맞춤)** | 이미지 전체가 보이도록 프레임 안에 비율 유지하며 축소 |
| **Fill (채우기)** | 프레임을 빈 공간 없이 이미지로 채움. 일부가 잘릴 수 있음 |
| **Stretch (늘리기)** | 프레임 크기에 맞게 이미지 비율을 무시하고 늘림. 비추천 |
| **Actual Size** | 이미지 원본 크기로 배치 |

> ✅ 교회 주보에서는 대부분 **Fill**을 사용합니다. 단체 사진처럼 이미지 전체가 보여야 할 때는 **Fit**을 사용하세요.

## 4. 이미지 크기·위치·자르기 제어

### 4.1 정확한 크기와 위치 입력

이미지 또는 프레임을 선택하면 상단 컨텍스트 도구 모음에 다음 필드가 표시됩니다.

| 필드 | 의미 | 사용 팁 |
| --- | --- | --- |
| `X` | 페이지 왼쪽 기준 수평 위치 | 정확한 정렬에 직접 입력 |
| `Y` | 페이지 상단 기준 수직 위치 | 정확한 정렬에 직접 입력 |
| `W` | 너비 | 비율 잠금 후 입력 |
| `H` | 높이 | 비율 잠금 후 입력 |
| 🔒 | 비율 잠금/해제 | ON: W 또는 H 변경 시 자동 비율 유지 |

### 4.2 비율 유지 크기 조절

- **드래그:** `Shift`를 누른 채 모서리 핸들 드래그
- **수치 입력:** 비율 잠금 아이콘을 켠 뒤 W 또는 H 중 하나만 입력

> ⚠️ 인물 사진과 로고는 비율을 무시하고 늘리면 찌그러져 보입니다. 항상 비율 잠금 상태에서 조절하세요.

### 4.3 이미지 회전

- 이미지 모서리 바깥쪽에서 회전 커서가 나타나면 드래그
- 컨텍스트 도구 모음의 회전 필드에 각도 입력
- `Layer → Transform → Rotate 90° CW/CCW`

### 4.4 이미지 반전

- 수평 반전: `Layer → Transform → Flip Horizontal`
- 수직 반전: `Layer → Transform → Flip Vertical`

## 용어 정리

| 용어 | 설명 |
| --- | --- |
| **Place** | 외부 이미지 파일을 문서에 가져오는 명령. `File → Place`. |
| **Picture Frame** | 이미지가 들어갈 공간을 미리 예약하는 프레임 오브젝트. |

## FAQ

**Q. Picture Frame에 배치한 이미지가 프레임 크기에 맞지 않아 빈 공간이 생깁니다.**

A. 프레임 선택 후 컨텍스트 도구 모음에서 **Fill**을 선택하세요. 이미지가 프레임을 빈 공간 없이 채웁니다. 단, 일부 영역이 잘릴 수 있으므로 프레임을 더블클릭한 뒤 이미지 위치를 조정하세요.

## 실수 방지 Tips

### 💡 Tip 1. 이미지는 항상 Picture Frame에 담아 사용하자

이미지를 Picture Frame 없이 직접 배치하면 크기와 위치를 조절할 때 이미지 자체가 변형되기 쉽습니다. Picture Frame을 사용하면 프레임 크기는 레이아웃 기준으로 고정하고, 프레임 안의 이미지만 독립적으로 조절할 수 있습니다.

### ⚠️ 자주 하는 실수

| 실수 | 결과 | 예방책 |
| --- | --- | --- |
| 이미지를 비율 무시하고 늘림 | 사람이나 로고가 찌그러짐 | 항상 비율 잠금 후 크기 조절 |
