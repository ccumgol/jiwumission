## 1. 그래픽 요소의 역할 — 교회 주보에서 이미지와 도형이 필요한 이유

교회 주보에서 텍스트만으로는 전달하기 어려운 내용이 있습니다. 이미지와 도형은 다음의 역할을 합니다:

| 역할 | 예시 |
| --- | --- |
| **시각적 계층 강조** | 배경 색상 띠로 섹션을 나눔 |
| **정보 전달 보완** | 예배당 위치 안내 지도, QR 코드 |
| **브랜드 아이덴티티 구현** | 교회 로고, 색상 테마 |
| **감성적 연결** | 예배 분위기의 사진, 계절 이미지 |
| **구분선 및 장식** | 가는 수평선, 모서리 장식 |

Layout Studio에서 그래픽 요소를 다루는 핵심 워크플로우는 다음과 같습니다:

`[이미지/그래픽 파일] → Place로 배치 → Picture Frame으로 제어
        ↓
[도형] → Shape Tool로 그리기 → Fill/Stroke로 색상 설정
        ↓
[레이어 패널] → 순서·그룹·잠금 관리
        ↓
[Align 패널] → 정렬·배분으로 레이아웃 완성`

## 2. 이미지 배치하기 — Place 명령 완전 정복

### 2.1 지원 이미지 파일 형식

Affinity Layout Studio는 다음 형식을 지원합니다:

| 형식 | 특징 | 인쇄 적합성 |
| --- | --- | --- |
| **TIFF (.tif)** | 무손실, 고품질, 레이어 지원 | ✅ 최우선 권장 |
| **PDF (.pdf)** | 벡터+래스터 혼합 가능 | ✅ 우수 |
| **PSD (.psd)** | Photoshop 레이어 파일 | ✅ 우수 |
| **PNG (.webp)** | 무손실, 투명 배경 지원 | ✅ 적합 (300DPI 이상) |
| **JPEG (.jpg)** | 손실 압축, 파일 경량 | ⚠️ 고화질 원본만 |
| **SVG (.svg)** | 벡터, 무한 확대 가능 | ✅ 로고·아이콘에 최적 |
| **EPS (.eps)** | 벡터, 레거시 형식 | ✅ 적합 |

> ⚠️ **JPEG 주의사항:** JPEG는 저장할 때마다 품질이 저하됩니다. 교회 로고나 중요 이미지는 반드시 PNG 또는 TIFF로 보관하세요.

### 2.2 Place 명령으로 이미지 배치

**경로:**

- `File` → `Place` (단축키: `⌘ Cmd + D` / `Ctrl + D`)

**단계별 방법:**

1. `File` → `Place` 선택
2. 파일 탐색 창에서 이미지 파일 선택 → `Open`
3. 커서가 이미지 불러오기 모드로 변경됨 (썸네일 + 십자 커서)
4. 다음 중 하나 선택:
  - **클릭:** 이미지 원본 크기로 배치 (크기 자동)
  - **드래그:** 드래그한 영역 크기에 맞게 배치
  - **기존 Picture Frame 클릭:** 해당 프레임 안에 이미지 배치

### 2.3 여러 이미지 한 번에 배치

1. `File` → `Place`
2. 파일 탐색 창에서 `⌘ Cmd + 클릭` (Mac) / `Ctrl + 클릭` (Win) 으로 여러 파일 선택
3. `Open` → 커서에 여러 이미지가 스택으로 로드됨
4. 클릭할 때마다 하나씩 배치 (남은 이미지 수가 커서에 표시됨)

> 💡 **드래그 앤 드롭도 가능합니다:**
>
> - 🍎 **Mac:** Finder에서 이미지 파일을 Affinity 문서로 드래그
> - 🖥️ **Windows:** 파일 탐색기에서 이미지 파일을 Affinity 문서로 드래그

### 2.4 Linked(링크) vs Embedded(임베드)

이미지를 배치할 때 두 가지 저장 방식 중 하나가 선택됩니다:

| 방식 | 설명 | 파일 크기 | 공유 시 |
| --- | --- | --- | --- |
| **Linked (링크)** | 이미지 파일이 외부에 존재, Affinity가 참조 | 작음 | 이미지 파일도 함께 전달 필요 |
| **Embedded (임베드)** | 이미지가 Affinity 파일 내부에 복사됨 | 커짐 | 파일 하나로 자체 완결 |

**기본 동작:** Place 명령 시 기본적으로 **Linked** 방식으로 배치됩니다.

**임베드로 전환:**

1. `Document` → `Resource Manager` 열기
2. 임베드할 이미지 선택
3. `Embed` 버튼 클릭

> 💡 **교회 주보 권장:**
>
> - **작업 중:** Linked (파일 크기 관리)
> - **인쇄소 제출 전:** 모든 이미지 Embedded 또는 Package 기능으로 이미지 폴더 함께 묶기

## 3. Picture Frame Tool — 이미지 프레임 활용

Picture Frame은 이미지가 들어갈 **공간을 미리 예약**하는 사각형 또는 타원형 박스입니다. 이미지를 직접 배치하는 것보다 더 정교한 제어가 가능합니다.

### 3.1 Picture Frame Tool의 종류

왼쪽 도구 모음에서 Picture Frame 관련 도구를 길게 누르면 하위 도구가 펼쳐집니다:

| 도구 이름 | 설명 |
| --- | --- |
| **Picture Frame Rectangle** | 사각형 이미지 프레임 |
| **Picture Frame Ellipse** | 원형/타원형 이미지 프레임 |

> 💡 사각형과 타원형 외의 모양(예: 별, 다각형, 자유 형태)으로 이미지 프레임을 만들려면:
> 도형을 먼저 그린 다음 `Layer` → `New Picture Frame from Shape`를 사용합니다.

### 3.2 Picture Frame 생성

1. 도구 모음에서 **Picture Frame Rectangle** 선택
2. 문서에서 드래그하여 프레임 생성 (빗금이 채워진 박스로 표시됨)
3. 프레임이 비어 있는 상태 = 이미지 대기 상태

### 3.3 Picture Frame에 이미지 배치

**방법 1 — Place로 배치:**

1. `File` → `Place` 실행
2. 이미지 선택 후 Picture Frame 위 클릭 → 프레임 안에 배치됨

**방법 2 — 드래그 앤 드롭:**
파일 탐색기에서 이미지를 Picture Frame 위로 드래그 → 자동으로 프레임 안에 배치됨

### 3.4 프레임과 이미지 각각 제어하기

Picture Frame 안에 이미지가 들어간 상태에서:

| 목적 | 방법 |
| --- | --- |
| **프레임 크기/위치 조절** | Move Tool(`V`)로 프레임 클릭 → 핸들 드래그 |
| **이미지 위치 조절** (프레임 내) | 프레임 더블클릭 → 이미지 이동 모드 진입 → 드래그 |
| **이미지 크기 조절** (프레임 내) | 프레임 더블클릭 → 이미지 선택 → 핸들 드래그 |
| **이미지 편집 모드 종료** | `Escape` 키 |

`[프레임 선택 상태] → 파란 테두리, 핸들 표시 → 프레임 이동/크기 조절
       ↓ 더블클릭
[이미지 편집 모드] → 주황 테두리, 이미지 핸들 표시 → 이미지 위치/크기 조절
       ↓ Escape
[프레임 선택 상태]`

### 3.5 이미지 채우기 방식 (Fit/Fill)

컨텍스트 도구 모음에서 다음 옵션을 선택할 수 있습니다:

| 옵션 | 설명 |
| --- | --- |
| **Fit (맞춤)** | 이미지 전체가 보이도록 프레임 안에 비율 유지하며 축소 (여백 발생 가능) |
| **Fill (채우기)** | 프레임을 빈 공간 없이 이미지로 채움 (비율 유지, 일부 잘릴 수 있음) |
| **Stretch (늘리기)** | 프레임 크기에 맞게 이미지 비율 무시하고 늘림 (비추천) |
| **Actual Size** | 이미지 원본 크기로 배치 |

> ✅ **교회 주보 권장:** 대부분의 경우 **Fill** 사용. 이미지 전체가 보여야 하는 경우(단체 사진 등) **Fit** 사용.

## 4. 이미지 크기·위치·자르기 제어

### 4.1 컨텍스트 도구 모음에서 정확한 크기/위치 입력

이미지(또는 프레임)를 선택하면 상단 컨텍스트 도구 모음에 다음 필드가 표시됩니다:

| 필드 | 의미 | 사용 팁 |
| --- | --- | --- |
| `X` | 페이지 왼쪽 기준 수평 위치 | 정확한 정렬에 직접 입력 |
| `Y` | 페이지 상단 기준 수직 위치 | 정확한 정렬에 직접 입력 |
| `W` | 너비 | 비율 잠금 후 입력 |
| `H` | 높이 | 비율 잠금 후 입력 |
| 🔒 | 비율 잠금/해제 | ON: W 또는 H 변경 시 자동 비율 유지 |

### 4.2 비율 유지 크기 조절

- **드래그 방식:** `Shift` 누른 채 모서리 핸들 드래그
- **수치 입력 방식:** 컨텍스트 도구 모음에서 🔒 비율 잠금 아이콘 ON → W 또는 H 중 하나만 입력

> ⚠️ **교회 인물 사진 특히 주의:** 인물 사진을 비율 무시하고 늘리면 사람이 찌그러져 보입니다. 항상 비율 잠금 상태에서 크기를 조절하세요.

### 4.3 이미지 회전

- **Move Tool 상태:** 이미지 모서리에서 약간 바깥으로 커서를 이동하면 회전 커서로 변경 → 드래그로 회전
- **정확한 각도:** 컨텍스트 도구 모음의 회전 필드(°)에 값 입력
- **90도 회전:** 컨텍스트 도구 모음의 회전 버튼 또는 `Layer` → `Transform` → `Rotate 90° CW/CCW`

### 4.4 이미지 반전 (Flip)

- 수평 반전: `Layer` → `Transform` → `Flip Horizontal`
- 수직 반전: `Layer` → `Transform` → `Flip Vertical`
- 또는 컨텍스트 도구 모음의 반전 아이콘

## 5. 이미지 교체 및 Resource Manager 활용

### 5.1 Resource Manager 열기

`Document` → `Resource Manager`

또는:
`Window` → `Resource Manager`

### 5.2 Resource Manager 기능

Resource Manager는 문서에 배치된 모든 이미지를 한 곳에서 관리합니다:

`┌─────────────────────────────────────────────────────┐
│  RESOURCE MANAGER                                   │
├──────────────┬──────────────┬───────────┬───────────┤
│ 파일명        │ 페이지       │ 상태      │ DPI       │
├──────────────┼──────────────┼───────────┼───────────┤
│ logo.webp     │ Master       │ Embedded  │ 300       │
│ photo1.jpg   │ Page 1       │ Linked ✅ │ 300       │
│ photo2.jpg   │ Page 3       │ Missing ❌│ —         │
│ banner.tif   │ Page 1       │ Linked ✅ │ 150 ⚠️   │
└──────────────┴──────────────┴───────────┴───────────┘`

| 상태 아이콘 | 의미 | 조치 |
| --- | --- | --- |
| ✅ (녹색) | 정상 연결 | — |
| ❌ (빨간) | 파일 없음 (이동/삭제됨) | `Replace`로 새 경로 지정 |
| ⚠️ (노랑) | 파일이 수정되었거나 DPI 부족 | `Update` 또는 원본 교체 |

### 5.3 누락된 이미지 재연결

1. Resource Manager에서 Missing(❌) 파일 선택
2. **`Replace`** 버튼 클릭
3. 파일 탐색창에서 원본 파일 또는 새 파일 선택 → `Open`

### 5.4 이미지 교체 (Replace Image)

레이아웃을 유지하면서 이미지 내용만 바꾸고 싶을 때:

**방법 1 — Resource Manager:**

1. `Document` → `Resource Manager`
2. 교체할 이미지 선택 → `Replace` → 새 이미지 선택

**방법 2 — 직접 교체:**

1. 교체할 이미지(또는 프레임) 선택
2. 컨텍스트 도구 모음에서 **"Replace Image"** 아이콘 클릭
3. 새 이미지 파일 선택

### 5.5 DPI 확인 — 인쇄 품질 체크

Resource Manager에서 각 이미지의 배치 DPI(Placed DPI)를 확인할 수 있습니다:

| DPI 범위 | 인쇄 품질 | 조치 |
| --- | --- | --- |
| **300 이상** | ✅ 양호 | — |
| **150~299** | ⚠️ 주의 | 가능하면 고해상도 원본 교체 |
| **150 미만** | ❌ 불량 | 반드시 교체 또는 크기 축소 |

> 💡 **배치 DPI vs 원본 DPI:**
> 이미지를 크게 늘리면 배치 DPI가 낮아집니다. 예를 들어 600DPI 이미지를 2배 크기로 늘리면 배치 DPI는 300DPI가 됩니다. 이미지를 너무 크게 확대하지 않도록 주의하세요.

### 6.1 주요 도형 도구

왼쪽 도구 모음에서 도형 그룹을 길게 누르면 다음 도구가 나타납니다:

| 도구 | 단축키 | 설명 |

| **Rectangle Tool** | `M` | 사각형 그리기. `Shift`+드래그: 정사각형 |
| **Ellipse Tool** | `L` | 타원 그리기. `Shift`+드래그: 정원 |

| **Triangle Tool** | — | 삼각형 |
| **Polygon Tool** | — | 다각형 (꼭짓점 수 설정 가능) |
| **Star Tool** | — | 별 모양 |
| **Line Tool** | — | 직선 |
| **Pen Tool** | `P` | 자유 벡터 패스 그리기 |

### 6.2 도형 그리기 기본

1. 원하는 도형 도구 선택
2. 문서에서 **드래그** → 도형 생성
3. 크기 조절: 핸들 드래그 또는 컨텍스트 도구 모음 W/H 입력

**유용한 드래그 수식어:**

| 수식어 | 효과 |
| --- | --- |
| `Shift` + 드래그 | 정비율 도형 (정사각형, 정원 등) |
| `⌥ Option` / `Alt` + 드래그 | 중심점에서 바깥으로 확장하며 그리기 |
| `Shift + ⌥ Option` / `Shift + Alt` | 정비율 + 중심점 기준 동시 적용 |

### 6.3 선 그리기 (장식선/구분선)

교회 주보에서 섹션을 나누는 가는 수평선은 Line Tool 또는 Rectangle Tool로 만듭니다:

**방법 1 — Line Tool:**

1. Line Tool 선택 (도형 그룹에서)
2. `Shift` 누른 채 가로로 드래그 → 완전 수평선 생성
3. Stroke 패널에서 선 두께 설정 (예: 0.5pt)

**방법 2 — 얇은 Rectangle (권장):**

1. Rectangle Tool(`M`) 선택
2. 얇은 사각형 드래그 (H: 0.5~1pt)
3. Fill 색상 설정, Stroke 없음으로 설정

- 이 방법이 더 안정적으로 인쇄됩니다.

### 6.4 모서리 둥글게 하기

사각형을 선택한 후:

- **컨텍스트 도구 모음:** 모서리 반경 필드에 값 입력 (예: 0.1")
- **직접 드래그:** 사각형 내부의 빨간 동그라미 핸들을 드래그
- 각 모서리 개별 설정: 컨텍스트 도구 모음의 잠금 아이콘 해제 후 각각 입력

## 7. 색상, 채우기, 선 (Fill & Stroke) 설정

### 7.1 Fill (채우기) 설정

도형이나 이미지 프레임의 배경 색상을 설정합니다.

**방법 1 — Colour 패널:**

- `View` → `Studio` → `Colour` 패널 열기
- 도형 선택 후 Colour 패널에서 색상 선택

**방법 2 — 컨텍스트 도구 모음:**

- 도형 선택 시 상단 컨텍스트 도구 모음에 채우기 색상 스왓치 표시
- 스왓치 클릭 → 색상 피커 열림

**CMYK 색상 직접 입력:**

1. Colour 패널에서 `CMYK` 모드 선택
2. C, M, Y, K 각 슬라이더 또는 수치 필드에 입력

> 📌 **교회 브랜드 색상 저장:**
> 자주 사용하는 색상을 Swatches 팔레트에 저장해두면 매번 수치를 입력하지 않아도 됩니다.
> Colour 패널 → Swatches 탭 → `+` 아이콘으로 현재 색상 저장

### 7.2 채우기 유형

| 유형 | 설명 | 사용 예 |
| --- | --- | --- |
| **Solid (단색)** | 단일 색상 채우기 | 배경 띠, 구분선 |
| **Linear Gradient** | 가로/세로 방향으로 색이 변하는 그라데이션 | 배경 장식 |
| **Radial Gradient** | 중심에서 바깥으로 퍼지는 그라데이션 | 원형 장식 요소 |
| **None (없음)** | 투명 배경 | 테두리만 있는 도형 |

**그라데이션 설정:**

1. 도형 선택
2. 도구 모음에서 **Fill Tool** 선택 (`F` 단축키)
3. 컨텍스트 도구 모음에서 그라데이션 유형 선택
4. 그라데이션 핸들로 방향/범위 조정
5. 색상 정류장(Color Stop)을 클릭하여 색상 변경

### 7.3 Stroke (선/테두리) 설정

도형의 외곽선을 설정합니다.

**Stroke 패널 열기:**

- `View` → `Studio` → `Stroke`

**주요 설정:**

| 설정 | 설명 |
| --- | --- |
| **Width (두께)** | 선 굵기 (pt 또는 픽셀) |
| **Style (스타일)** | 실선, 점선, 파선 등 |
| **Caps** | 선 끝 모양 (Butt/Round/Square) |
| **Join** | 꺾이는 부분 모양 (Miter/Round/Bevel) |
| **Align** | 선 위치 (Inside/Center/Outside) |

> 📌 **인쇄 시 얇은 선 주의:**
> 0.25pt 미만의 선은 일부 인쇄기에서 재현되지 않을 수 있습니다. 가이드라인 목적의 선은 0.25~0.5pt를 권장합니다.

### 7.4 불투명도 (Opacity) 설정

- 이미지나 도형을 선택한 상태에서 Layers 패널 상단의 **Opacity** 슬라이더 조절
- 0% = 완전 투명, 100% = 완전 불투명

**사용 예:** 배경 사진 위에 반투명 색상 오버레이를 올려 텍스트 가독성 확보

### 7.5 블렌드 모드 (Blend Mode)

도형이나 이미지가 아래 레이어와 혼합되는 방식을 설정합니다.

**설정 위치:** Layers 패널 상단 드롭다운 (`Normal`이 기본값)

교회 주보에서 자주 사용하는 블렌드 모드:

| 모드 | 효과 | 사용 예 |
| --- | --- | --- |
| **Normal** | 기본값, 혼합 없음 | 대부분의 경우 |
| **Multiply** | 색상이 어두워짐 | 배경 사진 위에 어두운 오버레이 |
| **Screen** | 색상이 밝아짐 | 어두운 배경에 밝은 요소 |
| **Overlay** | 대비가 강해짐 | 텍스처 효과 |

## 8. 레이어 패널 활용 — 오브젝트 관리의 핵심

### 8.1 Layers 패널 열기

- `View` → `Studio` → `Layers`

### 8.2 Layers 패널 구조

`┌──────────────────────────────────┐
│  LAYERS                   ⚙️     │
├──────────────────────────────────┤
│  👁 🔒  [그룹: Header]           │ ← 레이어 그룹
│    👁 🔒  [Rectangle: 배경띠]    │
│    👁 🔒  [Image: logo.webp]      │
│  👁 🔒  [Text: 설교 제목]        │
│  👁 🔒  [Image: photo1.jpg]      │
│  👁 🔒  [Text: 본문]             │
│  👁 🔒  [A-Master (2 pages)]     │ ← 마스터 페이지 레이어
└──────────────────────────────────┘
  👁 = 가시성 토글  🔒 = 잠금 토글`

### 8.3 레이어 순서 변경 (Z-순서)

레이어 패널에서 **위에 있는 항목이 화면 앞쪽(위층)**에 표시됩니다.

- **드래그:** 레이어 항목을 위아래로 드래그하여 순서 변경
- **메뉴:** `Arrange` → `Bring to Front / Send to Back / Bring Forward / Send Backward`
- **단축키:**

| 기능 | Mac | Windows |
| --- | --- | --- |
| 맨 앞으로 | `⌘ Cmd + Shift + ]` | `Ctrl + Shift + ]` |
| 앞으로 한 칸 | `⌘ Cmd + ]` | `Ctrl + ]` |
| 뒤로 한 칸 | `⌘ Cmd + [` | `Ctrl + [` |
| 맨 뒤로 | `⌘ Cmd + Shift + [` | `Ctrl + Shift + [` |

### 8.4 레이어 가시성 및 잠금

| 아이콘 | 기능 |
| --- | --- |
| 👁 (눈 아이콘) | 클릭하여 레이어 숨기기/표시 토글 |
| 🔒 (자물쇠) | 클릭하여 레이어 잠금/해제 — 잠긴 레이어는 선택 불가 |

> 💡 **마스터 페이지 레이어 잠금:** 마스터 페이지에서 온 요소들은 자동으로 잠겨 있습니다. 의도치 않게 클릭하여 이동하는 사고를 방지합니다.

### 8.5 레이어 그룹화 및 해제

관련 요소들을 그룹으로 묶으면 함께 이동하고 관리하기 편합니다.

**그룹화:**

1. 묶을 레이어들을 선택 (Shift + 클릭 또는 드래그 선택)
2. 단축키: `⌘ Cmd + G` / `Ctrl + G`
3. 또는 `Arrange` → `Group`

**그룹 해제:**

- 단축키: `⌘ Cmd + Shift + G` / `Ctrl + Shift + G`

**그룹 안으로 진입:**

- 그룹 더블클릭 → 그룹 내 개별 요소 선택 가능

### 8.6 레이어 이름 변경

레이어가 많아지면 이름 지정이 필수입니다:

1. 레이어 이름 더블클릭
2. 새 이름 입력 (예: `배경-색상띠`, `사진-예배`, `텍스트-설교제목`)
3. `Enter` 확정

## 9. 정렬 및 배분 (Align & Distribute)

여러 오브젝트를 정확하게 정렬하고 일정한 간격으로 배분합니다.

### 9.1 Alignment 패널 열기

- `View` → `Studio` → `Alignment`
- 또는 여러 오브젝트 선택 시 컨텍스트 도구 모음에 정렬 아이콘 표시

### 9.2 정렬 옵션

**정렬 기준 선택:**
패널 상단 드롭다운에서 기준 선택:

- **Selection (선택 영역 기준):** 선택된 오브젝트들의 경계 박스 기준
- **Page (페이지 기준):** 페이지 전체 기준
- **Spread (스프레드 기준):** 마주보는 페이지 전체 기준
- **Key Object (기준 오브젝트):** 특정 오브젝트를 기준으로 나머지를 정렬

**수평 정렬:**

| 아이콘 | 기능 |
| --- | --- |
| ⊞◀ | 왼쪽 가장자리 정렬 |
| ⊞◆ | 가로 중심 정렬 |
| ⊞▶ | 오른쪽 가장자리 정렬 |

**수직 정렬:**

| 아이콘 | 기능 |
| --- | --- |
| ⊟▲ | 위쪽 가장자리 정렬 |
| ⊟◆ | 세로 중심 정렬 |
| ⊟▼ | 아래쪽 가장자리 정렬 |

### 9.3 배분 옵션 (Distribute)

오브젝트 간의 간격을 균등하게 배분합니다.

| 옵션 | 기능 |
| --- | --- |
| **수평 균등 배분** | 오브젝트들의 가로 간격을 동일하게 |
| **수직 균등 배분** | 오브젝트들의 세로 간격을 동일하게 |
| **수평 간격 동일** | 오브젝트들 사이의 빈 공간을 동일하게 |
| **수직 간격 동일** | 오브젝트들 사이의 빈 공간을 동일하게 |

### 9.4 스냅 기능 (Snapping)

오브젝트를 이동할 때 가이드, 다른 오브젝트, 그리드 등에 자동으로 달라붙게 합니다.

**스냅 토글:**

- 상단 메뉴 바의 자석 아이콘 클릭
- 또는 단축키: `;` 키

**스냅 대상 설정:**

- `View` → `Snapping Options` → 스냅할 대상 선택:
  - Snap to Guides (가이드라인)
  - Snap to Grid (그리드)
  - Snap to Object Bounding Box (오브젝트 경계)
  - Snap to Object Geometry (오브젝트 형태)

3. 또는 스튜디오 전환 탭에서 `Canva AI` 클릭

### 10.2 주요 AI 기능

### Generative Fill (생성형 채우기) 💳

이미지의 특정 영역을 AI로 생성한 콘텐츠로 대체합니다.

**사용 방법:**

1. 편집할 이미지 선택
2. `Canva AI Studio` → `Generative Fill`
3. 붓으로 변경할 영역 선택
4. 텍스트 프롬프트 입력 (예: "blue sky with clouds")
5. `Generate` 클릭

**교회 주보 활용 예:**

- 예배 사진 배경에서 불필요한 요소 제거
- 이미지 배경을 단색으로 교체

### Generative Expand (이미지 확장) 💳

이미지가 프레임보다 작을 때 AI로 주변을 자연스럽게 채웁니다.

**사용 방법:**

1. 이미지 선택 → `Canva AI Studio` → `Generative Expand`
2. 캔버스 경계를 원하는 크기로 드래그하여 확장
3. `Expand` 클릭 → AI가 빈 영역을 자동으로 채움

**교회 주보 활용 예:**

- 16:9 사진을 주보 표지의 가로 비율에 맞게 확장
- 세로 사진을 가로로 늘여 배경 이미지로 활용

### Remove Background (배경 제거) 💳

사진의 배경을 자동으로 제거하여 피사체만 남깁니다.

**사용 방법:**

1. 이미지 선택 → `Canva AI Studio` → `Remove Background`
2. AI가 자동으로 배경 제거
3. 필요시 브러시 도구로 세부 조정

**교회 주보 활용 예:**

- 목사님 사진을 배경 없이 투명 PNG로 만들어 디자인에 자연스럽게 합성
- 로고 이미지 배경 제거

### Super Resolve (고해상도 업스케일) 💳

낮은 해상도 이미지를 AI로 선명하게 고해상도로 변환합니다.

**교회 주보 활용 예:**

- 오래된 교회 사진이나 스마트폰으로 찍은 작은 이미지를 인쇄 품질로 업스케일

| 용어 | 설명 |

| **Resource Manager** | 문서의 모든 이미지 파일을 관리하는 패널. DPI 확인, 교체, 링크 수정 가능. |

| **Fit (맞춤)** | 이미지가 프레임 안에 전체가 보이도록 맞추는 채우기 방식. |
| **Fill (채우기, 이미지)** | 이미지가 프레임을 빈 공간 없이 채우는 방식. 일부 잘릴 수 있음. |
| **Text Wrap (텍스트 래핑)** | 이미지·도형 주변으로 텍스트가 흐르도록 설정. Chapter 5에서 상세 설명. |
| **Generative Fill** | Canva AI 기능. 이미지 특정 영역을 AI 생성 콘텐츠로 교체. 유료. |
| **Super Resolve** | Canva AI 기능. 저해상도 이미지를 고해상도로 업스케일. 유료. |

## 13. FAQ

**Q1. 이미지를 배치했는데 화면에서는 선명한데 인쇄하면 흐릿하게 나옵니다.**

A. 원인은 이미지 해상도(DPI) 부족입니다. 확인 단계:

1. `Document` → `Resource Manager`에서 해당 이미지의 `Placed DPI` 확인
2. 300 DPI 미만이면 더 고해상도의 원본 이미지로 교체하거나, 이미지 표시 크기를 줄임
3. 화면에서는 72~96DPI도 선명해 보이지만, 인쇄에는 300DPI가 필요합니다

---

**Q2. Picture Frame에 배치한 이미지가 프레임 크기에 맞지 않아 빈 공간이 생깁니다.**

A. 채우기 방식을 변경하세요:

1. 이미지 프레임 선택 (파란 테두리)
2. 컨텍스트 도구 모음에서 **`Fill`** 옵션 클릭
3. 이미지가 빈 공간 없이 프레임을 채움 (일부 잘릴 수 있음)

이미지 위치가 마음에 들지 않으면: 프레임 더블클릭 → 이미지를 드래그하여 원하는 부분이 보이도록 조정.

---

**Q3. Resource Manager에서 Missing(빨간) 표시가 나타납니다.**

A. 이미지 원본 파일의 경로가 변경된 것입니다. 해결 방법:

1. Resource Manager에서 해당 이미지 선택
2. **`Replace`** 버튼 클릭
3. 원본 파일의 새 위치를 탐색하여 선택

파일을 자주 이동하는 경우: Affinity 파일과 이미지 폴더를 같은 상위 폴더 안에 두고, 상대 경로가 유지되도록 관리하면 Missing 오류를 줄일 수 있습니다.

---

**Q4. 로고를 배치했는데 흰색 배경이 함께 나타납니다.**

A. 로고 파일의 배경이 투명하지 않기 때문입니다. 해결 방법:

1. **가장 좋은 방법:** 로고 원본 파일을 **PNG (투명 배경)** 또는 **SVG** 형식으로 저장하여 재배치
2. **Canva AI (유료):** `Remove Background` 기능으로 배경 제거 후 PNG로 내보내기
3. **Pixel Studio 활용:** 이미지를 Pixel Studio에서 열어 직접 배경 마스크 처리

---

**Q5. 도형과 텍스트를 함께 그룹으로 묶었는데 그룹 안의 텍스트를 편집할 수 없습니다.**

A. 그룹 안의 요소를 편집하려면 그룹을 먼저 진입해야 합니다:

1. 그룹을 **더블클릭** → 그룹 편집 모드 진입
2. 원하는 텍스트 프레임 클릭 → 선택
3. 텍스트 더블클릭 → 텍스트 편집 모드
4. `Escape` 두 번으로 그룹 편집 모드 → 전체 선택 모드 순으로 복귀

---

**Q6. 이미지를 정확히 페이지 중앙에 배치하고 싶습니다.**

A. Alignment 패널을 사용합니다:

1. 이미지 선택
2. `View` → `Studio` → `Alignment` 패널 열기
3. 정렬 기준 드롭다운에서 **`Page`** 선택
4. **가로 중심 정렬** 아이콘 클릭 + **세로 중심 정렬** 아이콘 클릭
5. 이미지가 페이지 정중앙에 배치됨

---

**Q7. 예배 사진 위에 텍스트를 올렸는데 텍스트가 잘 안 보입니다.**

A. 사진 위 텍스트 가독성을 높이는 방법 세 가지:

**방법 1 — 반투명 색상 오버레이 (가장 효과적):**

1. Rectangle Tool로 텍스트 영역과 같은 크기 사각형 생성
2. Fill: 검정색 (`#000000`) 또는 짙은 색상
3. Opacity: 40~60%
4. Layers 패널에서 이 사각형을 텍스트 아래, 사진 위로 배치

**방법 2 — 텍스트 색상을 흰색으로:**
텍스트 선택 → Character 패널에서 색상을 흰색(CMYK: 0/0/0/0)으로 변경

**방법 3 — 텍스트에 Drop Shadow 효과:**
텍스트 선택 → `Layer` → `Layer Effects` → `Drop Shadow` 활성화

## 14. Tips: 실수를 줄이고 더 잘 사용하는 법

### 💡 Tip 1: 이미지는 항상 Picture Frame에 담아 사용하자

이미지를 Picture Frame 없이 직접 배치하면 크기와 위치를 조절할 때 이미지 자체가 변형됩니다. Picture Frame을 먼저 만들어두면 프레임 크기는 레이아웃 기준으로 고정하고, 프레임 안의 이미지 위치만 독립적으로 조절할 수 있어 훨씬 유연하게 작업할 수 있습니다.

---

### 💡 Tip 2: 교회 브랜드 색상을 Swatches에 저장하자

교회 주보에 자주 쓰이는 브랜드 색상(CMYK 값)을 Swatches 팔레트에 저장해두면:

- 매번 CMYK 값 입력 불필요
- 실수로 잘못된 색상 사용 방지
- 팀원과 색상 공유 가능 (팔레트 내보내기)

**저장 방법:**

- Colour 패널 → Swatches 탭 → `+` 클릭 → 이름 지정 (예: `Brand Blue`, `Brand Gray`)

---

### 💡 Tip 3: 레이어 이름을 항상 지정하자

레이어가 10개 이상 쌓이면 `Rectangle`, `Image`, `Text` 등 기본 이름으로는 어떤 요소인지 구별이 안 됩니다. 배치 직후 레이어 이름을 지정하는 습관을 들이세요:

`좋은 이름 예시:
- "배경-상단띠-블루"
- "사진-예배당-표지"
- "텍스트-설교제목"
- "QR-홈페이지"`

---

### 💡 Tip 4: 중요한 배경 요소는 레이어 잠금을 걸어두자

마스터 페이지에서 온 배경 요소 외에도, 일반 페이지의 배경 이미지나 색상 띠처럼 실수로 건드리면 안 되는 요소는 Layers 패널에서 🔒 잠금 처리하세요. 텍스트나 이미지 배치 작업 시 실수로 배경을 이동하는 사고를 방지합니다.

---

### 💡 Tip 5: 인쇄 전 Resource Manager로 DPI 일괄 점검하자

PDF 내보내기(Chapter 8) 전에 `Document → Resource Manager`를 열어 모든 이미지의 DPI를 한 번에 확인하는 것을 루틴으로 만드세요. 300DPI 미만 이미지가 있으면 교체하거나 크기를 줄여 DPI를 올리세요.

---

### 💡 Tip 6: 정렬은 Align 패널, 미세 조정은 수치 입력으로

마우스 드래그로 여러 요소를 맞추려 하면 시간이 오래 걸리고 부정확합니다:

- **대략적 위치 잡기:** 드래그 + 스냅
- **섹션 단위 정렬:** Alignment 패널의 정렬/배분 버튼
- **픽셀 단위 정확도:** 컨텍스트 도구 모음의 X/Y/W/H 수치 직접 입력
