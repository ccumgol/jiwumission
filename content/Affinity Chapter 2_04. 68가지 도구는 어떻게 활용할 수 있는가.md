📌 Affinity 3의 68가지 도구는 어떻게 활용할 수 있는가?

Affinity 3는 벡터, 픽셀, 레이아웃 스튜디오별로 다양한 도구를 제공하며, 각 스튜디오의 특성에 맞춰 이동, 선택, 그리기, 편집, 채우기, 텍스트, 측정 등 광범위한 기능을 수행할 수 있습니다.

Affinity 3로 전환을 고민하는 디자이너를 위해 **68개 툴의 모든 기능**을 꼼꼼하게 해부한 궁극의 가이드입니다. 포토샵이나 일러스트레이터와 비교하며 **Vector Studio, Pixel Studio, Layout Studio** 각 영역별 툴의 **실제 사용법과 차이점**을 명확히 설명하여, 새로운 툴 환경에 빠르게 적응하고 **작업 효율을 극대화**할 수 있는 실질적인 인사이트를 제공합니다.

타임라인 요약: Affinity 3의 68개 툴 가이드 (Vector, Pixel, Layout 스튜디오)

본 요약은 Affinity 3로 전환을 고려하는 디자이너를 위해 **Vector Studio, Pixel Studio, Layout Studio** 세 가지 주요 스튜디오의 툴들을 원본 영상의 시간 순서에 따라 체계적으로 정리한 것입니다.

# 1. Vector Studio 툴 살펴보기 (Adobe Illustrator 동급)

Vector Studio는 Adobe Illustrator와 동등한 기능을 제공하는 작업 공간입니다.

1.1. 기본 이동 및 변형 툴

1. **Move Tool (이동 툴)**
  1. **기능**: 개체를 이동, 회전, 크기 조절에 사용되며, 가장 자주 사용될 툴로 예상됩니다.
  2. **회전**: Shift 키를 누른 채 회전하면 15도 단위로 스냅됩니다.
  3. **선택 해제**: 빈 공간을 클릭하거나 Esc 키를 눌러 해제할 수 있습니다.

1.2. 아트보드 및 도형 편집 툴

1. **Artboard Tool (아트보드 툴)**
  1. **기능**: 새로운 아트보드를 추가하거나 직접 그려서 아트보드를 생성할 수 있습니다.
  2. **크기 조정**: 생성 후 Transform 패널에서 크기를 사용자 지정할 수 있습니다.
  3. **정보 확인**: 툴 위에 마우스를 올리면 다양한 정보를 확인할 수 있으며, 대부분의 툴에는 키보드 단축키가 존재합니다.
2. **Node Tool (노드 툴)**
  1. **기능**: Illustrator의 Direct Selection Tool과 유사하며, 개별 앵커 포인트를 선택하고 수정하는 데 사용됩니다.
  2. **Live Shape 편집**: 기본 도형(Live Shape)의 경우, 개별 앵커 포인트를 편집하려면 먼저 **Curves(곡선)**로 변환해야 합니다.
  3. **활용**: 변환 후 노드 툴을 사용하여 앵커 포인트를 이동하거나 다양한 방식으로 조정할 수 있습니다.
3. **Point Transform Tool (포인트 변형 툴)**
  1. **기능**: 스케일 툴과 유사하게 작동하지만, 특정 포인트를 기준으로 크기를 조절하는 데 사용됩니다.
  2. **작동 방식**: 특정 포인트를 잡고 이동시키면 해당 지점을 기준으로 크기가 조절됩니다.
  3. **평가**: Illustrator의 스케일 툴과는 약간 다르며, 연습이 필요할 것으로 보입니다.

1.3. 모서리 및 경로 관련 툴

1. **Corner Tool (코너 툴)**
  1. **단축키**: 키보드에서 **C**를 누르면 활성화됩니다.
  2. **기능**: 개체의 개별 또는 여러 모서리 포인트를 선택하여 모서리를 둥글게 만들 수 있습니다.
2. **Contour Tool (컨투어 툴)**
  1. **단축키**: 키보드에서 **O**를 누르면 활성화됩니다.
  2. **기능**: Illustrator의 Path Offset과 유사하게, 도형을 더 크거나 작게 만들고 모서리를 둥글게 처리할 수 있습니다.
  3. **옵션**: 상단에 다양한 옵션이 제공되며, 작업 완료 후 **Bake Appearance**를 눌러 변형을 확정할 수 있습니다.
  4. **후속 작업**: 변형 확정 후에는 Node Tool을 사용하여 기하학적 구조의 개별 포인트를 조정할 수 있습니다.

1.4. 드로잉 및 브러시 툴

1. **Pen Tool (펜 툴)**
  1. **단축키**: 키보드에서 **P**를 누르면 활성화됩니다.
  2. **기능**: 사용자 지정 모양을 그리거나, 직선(점들을 연결) 또는 곡선(클릭 후 드래그)을 그릴 수 있습니다.
  3. **Rubber Band Mode**: Illustrator와 달리 곡선이 어떻게 보일지 미리 확인하기 어려운데, **Rubber Band Mode** 옵션을 켜면 곡선을 그리기 전에 미리 볼 수 있어 유용합니다.
2. **Pencil Tool (연필 툴)**
  1. **기능**: 거의 자유롭게 손으로 그릴 수 있습니다.
  2. **Stabilizer (안정기)**: 경로를 안정화시켜 마우스나 트랙패드로 그릴 때 자동으로 경로를 부드럽게(smooth) 처리해 주어 자유 드로잉을 더 쉽게 만듭니다.
3. **Path Brush Tool (패스 브러시 툴)**
  1. **기능**: Illustrator의 벡터 브러시처럼 벡터 기반의 브러시로 그릴 수 있습니다.
  2. **벡터 브러시**: 이 툴은 벡터 브러시이며, 실시간 미리보기가 제공되어 매우 빠르고 반응성이 좋습니다.
  3. **브러시 종류**: 아크릴(acrylics), 마커(markers) 등 다양한 브러시 세트를 선택하여 사용할 수 있습니다.

1.5. 자르기 및 폭 조절 툴

1. **Knife Tool (나이프 툴)**
  1. **기능**: Illustrator의 Knife Tool(채워진 도형 자르기)과 Scissors Tool(경로로 만들어진 도형 자르기)의 기능을 하나로 통합했습니다.
  2. **작동 방식**: 경로 위에 호버링하면 가위 아이콘으로 바뀌며 클릭 시 자국이 생기고, Node Tool로 해당 경로를 분리할 수 있습니다.
  3. **단축키**: 키보드에서 **K**를 누르면 활성화됩니다.
  4. **직선 자르기**: Shift 키를 누르면 직선으로 자를 수 있습니다.
2. **Width Tool (폭 툴)**
  1. **단축키**: 키보드에서 **W**를 누르면 활성화됩니다.
  2. **기능**: 그려진 선(경로)의 특정 지점을 클릭하여 해당 지점의 **폭(width)**을 조절할 수 있는 매우 간단하고 유용한 기능입니다.

1.6. 채우기 및 투명도 툴

1. **Fill Tool (채우기 툴)**
  1. **기능**: 도형에 다양한 종류의 채우기(Fill)를 추가할 수 있습니다.
  2. **채우기 유형**: 드롭다운 메뉴에서 **Solid(단색)** 또는 **Gradients(그라데이션)**를 선택할 수 있습니다.
  3. **그라데이션**: **Linear(선형)** 또는 **Radial(방사형)** 그라데이션을 지원하며, Photoshop의 그라데이션 툴과 유사하게 빠르고 반응성이 좋습니다. (Illustrator의 그라데이션 툴보다 훨씬 개선됨)
  4. **조정**: 슬라이더에서 지점을 선택하고 색상을 지정하며 중간 지점을 자유롭게 조정할 수 있습니다.
2. **Transparency Tool (투명도 툴)**
  1. **기능**: 한 색상에서 투명도로 페이드(fade) 효과를 줄 수 있으며, 색상 오버레이 등에 투명도를 빠르게 적용할 때 유용합니다.

1.7. 도형 및 빌더 툴

1. **Shape Tools (도형 툴)**
  1. **편의성**: Illustrator에서 직접 만들어야 했던 많은 **사전 설정된 모양(preset shapes)**이 제공되어 매우 편리합니다.
  2. **Cog Tool (톱니바퀴 툴)**: 톱니바퀴 모양을 생성하며, **Live Shape**이므로 톱니 크기, 노치 크기 등 톱니바퀴 관련 설정을 상단에서 세부 조정할 수 있습니다.
  3. **변환 시 주의**: 개별 앵커 포인트를 조정하려면 **Curves(곡선)**로 변환해야 하며, 이 경우 상단 옵션 바의 설정을 잃게 됩니다.
  4. **숨겨진 프리셋**: Cog Tool 아래에는 톱니바퀴가 아닌 다른 모양의 프리셋도 포함되어 있습니다.
    1. 예시: **Star spiky thing**이나 **Sunburst(선버스트)** 모양이 Cog Tool 내에 포함되어 있습니다.
2. **Shape Builder Tool (셰이프 빌더 툴)**
  1. **단축키**: 키보드에서 **S**를 누르면 활성화됩니다.
  2. **기능**: 여러 도형을 선택한 후, 상단에서 **추가(add)** 또는 **빼기(subtract)** 모드를 선택하여 드래그로 도형들을 결합하거나 영역을 깎아낼 수 있습니다.
  3. **활용**: 복잡한 디자인에서 도형을 결합할 때 유용합니다.
3. **Vector Flood Fill Tool (벡터 채우기 툴)**
  1. **기능**: 툴바 상단이나 하단에서 색상을 선택하여 영역을 채울 수 있는 매우 간단한 툴입니다.

1.8. 텍스트 및 배치 툴

1. **Artistic Text Tool (예술적 텍스트 툴)**
  1. **기능**: Illustrator의 Point Type과 유사하며, 클릭 후 타이핑하고 크기를 조절하여 제목 등에 사용하기 좋습니다.
2. **Frame Text Tool (프레임 텍스트 툴)**
  1. **기능**: Illustrator의 Area Text(단락 텍스트)와 같으며, 텍스트 영역을 지정하여 입력하면 줄 바꿈이 자동으로 적용됩니다.
  2. **조정**: 텍스트 프레임 크기를 조정하면 텍스트 줄 바꿈이 실시간으로 반영됩니다.
3. **Filler Text 삽입**: Text 메뉴 > Insert > Filler Text를 통해 자리 표시자 텍스트를 삽입할 수 있습니다.
4. **Place Tool (배치 툴)**
  1. **기능**: 컴퓨터에서 이미지를 선택하여 문서에 배치할 수 있습니다.
5. **Vector Crop Tool (벡터 자르기 툴)**
  1. **기능**: 그룹화된 아트워크(Command/Control + G)를 선택한 후 이 툴을 사용하면, 아트워크 바깥쪽의 여분을 잘라내어 **클린 엣지(clean edges)**를 만들 수 있습니다.
  2. **활용**: 패턴 제작 시 반복되거나 타일링되는 경우 유용합니다.
  3. **복구**: 자르기 후에도 아트워크가 손실되지 않으며, 다시 확장하여 조정할 수 있습니다.
6. **Measure Tool (측정 툴)**
  1. **기능**: 서로 다른 지점들을 측정할 수 있으며, 상단에 옵션이 표시됩니다.
  2. **제한**: 클릭을 해제하면 측정값이 사라지므로, 영구적으로 스탬프처럼 남기는 기능은 현재 지원되지 않는 것으로 보입니다.
7. **Area Tool (면적 툴)**
  1. **기능**: 기술 도면이나 다이어그램에서 **총 표면적**과 개별 세그먼트의 표면적을 측정할 수 있습니다.
8. **Color Picker Tool (색상 선택 툴)**
  1. **기능**: 확대 원을 통해 매우 구체적인 색상을 선택할 수 있으며, 선택된 색상이 현재 선택 색상이 됩니다. (Illustrator의 Eyedropper Tool과 혼동 주의)
9. **Style Picker Tool (스타일 선택 툴)**
  1. **기능**: 텍스트의 획(stroke) 두께나 색상 등 스타일 속성을 한 객체에서 다른 객체로 복사할 수 있습니다.
10. **추가 툴 접근**: 다른 스튜디오의 툴도 툴바에서 접근 가능하며, 툴바에서 드래그하여 작업 공간을 완전히 사용자 지정할 수 있습니다.

# 2. Pixel Studio 툴 살펴보기 (Affinity 버전 Photoshop)

Pixel Studio는 Affinity의 래스터 이미지 편집 환경입니다.

2.1. 이동 및 변형 툴

1. **Move Tool (이동 툴)**
  1. **기능**: 선택, 이동, 크기 조절, 회전이 가능합니다.
  2. **Shift 키**: Shift 키를 누르면 왜곡(distort) 없이 변형됩니다.
2. **Perspective Tool (원근 툴)**
  1. **기능**: 라이브 필터로 적용되며, 도형의 원근을 왜곡하고 조정할 수 있습니다.
  2. **Rasterization**: 이 툴을 포함한 일부 툴은 효과 적용을 위해 이미지 레이어를 **래스터화(rasterized)** 시킵니다.
  3. **Mesh Warp**: Perspective Warp보다 더 자유롭게 왜곡하고 변형할 수 있는 Mesh Warp 기능도 제공됩니다.
3. **Deform Tool (변형 툴)**
  1. **상태**: 사용법을 명확히 파악하지 못했으며, 현재는 건너뜁니다.

2.2. 캔버스 및 선택 툴

1. **Crop Tool (자르기 툴)**
  1. **제한**: 아트보드를 사용하는 경우 이 툴은 비활성화됩니다.
  2. **기능**: 캔버스를 자르는 데 사용됩니다.
  3. **조정**: Command/Control 키를 누르면 중앙에서부터 자를 수 있고, Shift 키를 누르면 비례적으로 크기 조절이 가능합니다.
  4. **격자 옵션**: 드롭다운 메뉴에서 **3분할 격자(thirds grid)** 또는 **황금 나선(golden spiral)** 구도를 적용하여 자를 수 있습니다.
2. **Selection Tools (선택 툴)**
  1. **Object Selection Tool (객체 선택 툴)**: 클릭하면 다른 객체들을 선택할 수 있습니다.
  2. **Selection Brush Tool (선택 브러시 툴)**: Photoshop의 Quick Selection Tool과 유사하며, 브러시로 칠하여 선택 영역을 추가하거나, Alt/Option 키를 눌러 선택 영역을 제거할 수 있습니다.
    1. **후속 작업**: 선택 후 레이어 마스크 등을 적용할 수 있습니다.
  3. **Flood Select Tool (영역 채우기 선택 툴)**: Photoshop의 Magic Wand Tool과 동일하며, 색상을 기반으로 선택합니다.
  4. **Marquee Tools (영역 선택 툴)**
    1. **Rectangular/Elliptical**: 사각형 또는 타원형 영역을 선택하며, Shift로 추가, Alt/Option으로 빼기가 가능합니다.
    2. **Column/Row**: 1픽셀 너비/높이의 열 또는 행을 선택하며, 옵션 바에서 1픽셀 이상의 값으로 너비를 변경할 수 있습니다.
  5. **Freehand Selection Tool (자유 선택 툴)**: 자유롭게 드로잉하여 선택합니다.
    1. **직선 선택**: Command/Control 키를 누른 상태에서 클릭하면 직선으로 점을 찍어 선택할 수 있습니다.

2.3. 채우기 및 브러시 툴 (래스터)

1. **Fill Tool (채우기 툴)**
  1. **작동 방식**: 이 툴을 드래그하면 레이어에 무언가가 추가되는데, 이는 Photoshop과 다르게 **새로운 채우기 레이어(new fill layer)**를 생성하는 방식입니다.
  2. **채우기 레이어**: 이 레이어에서 **Solid(단색)** 또는 **Linear(선형 그라데이션)**를 선택할 수 있습니다.
  3. **색상 선택**: 색상이 흰색으로 설정되어 있어 보이지 않을 수 있으므로, 툴바 하단에서 색상 선택기를 열어 색상을 변경해야 합니다.
  4. **버그 보고**: 초기 문서에서 이 기능에 버그가 있었으나, 새 문서를 만들고 레이어를 복사하여 해결했습니다.
  5. **사용법**: 채우기 레이어를 선택한 상태에서 그라데이션 유형을 지정하고 드래그하여 조정하며, 언제든지 레이어를 선택하여 그라데이션을 수정할 수 있습니다.
2. **Flood Fill Tool (영역 채우기 툴)**
  1. **기능**: Photoshop의 Paint Bucket Tool과 매우 유사하며, 새 픽셀 레이어에서 색상을 선택하여 채울 수 있습니다.
3. **Paintbrush Tool (페인트 브러시 툴)**
  1. **선호도**: 사용자가 가장 자주 사용하는 도구 중 하나입니다.
  2. **옵션**: 상단에서 불투명도(opacity), 흐름(flow), 경도(hardness) 등을 설정할 수 있습니다.
  3. **브러시 패널**: 우측의 브러시 패널에서 다양한 브러시를 선택할 수 있으며, Photoshop 브러시 파일도 문제없이 가져와 사용할 수 있습니다.
  4. **레이어 요구사항**: 이 툴은 **픽셀 레이어**에서만 브러싱이 가능하며, 이미지 레이어에서 사용하려면 래스터화해야 합니다.
4. **Paint Mixer Brush Tool (페인트 믹서 브러시 툴)**
  1. **기능**: 디지털 페인팅에서 **혼합 및 블렌딩**을 시뮬레이션하는 데 사용됩니다.
  2. **작동 방식**: 선택한 브러시 스타일로 칠하는 영역의 픽셀을 섞고 블렌딩합니다.
  3. **레이어 처리**: 이미지 레이어에서 사용 시 레이어가 래스터화됩니다.
  4. **설정**: 강도(strength) 등의 설정을 제어할 수 있습니다.

2.4. 파괴적 편집 툴 (Destructive Editing)

이 섹션의 툴들은 영구적으로 이미지를 손상시키는 **파괴적 편집(destructively working)** 방식으로 작동합니다.

1. **Color Replacement Tool (색상 교체 툴)**
  1. **평가**: 색상을 교체할 수 있지만, 파괴적으로 작동하며 더 나은 방법이 있다고 판단하여 무시합니다.
2. **Pixel Tool (픽셀 툴)**
  1. **기능**: 1픽셀 선을 그릴 수 있으나, 사용자가 흥미를 느끼지 못했습니다.
3. **Erase Brush Tool (지우개 브러시 툴)**
  1. **이미지 레이어**: 이미지 레이어에서 사용하면 자동으로 **레이어 마스크**가 생성되는 점이 마음에 든다고 언급합니다.
  2. **픽셀 레이어**: 픽셀 레이어에서는 이미지를 단순히 지웁니다.
4. **Background Erase Brush (배경 지우개 브러시)**
  1. **기능**: 배경을 선택하여 제거할 수 있으며, 설정 조정을 통해 더 나은 결과를 얻을 수 있습니다.
  2. **평가**: 영구적으로 배경을 지우므로, 전문적인 작업에는 의존하지 않을 툴로 평가됩니다.
5. **Flood Erase Tool (영역 지우기 툴)**
  1. **기능**: Paint Bucket Tool처럼 한 번의 클릭으로 배경을 제거합니다.
  2. **평가**: 급할 때 유용할 수 있으나, 주로 사용하지 않을 툴로 판단됩니다.

2.5. 조정 및 필터 브러시 툴

1. **Adjustment Brush Tool (조정 브러시 툴)**
  1. **기능**: 조정 레이어(Adjustment Layer)를 선택하고 브러시로 칠하여 해당 영역에 조정을 적용합니다.
  2. **기본 기능**: 기본적으로 영역을 **어둡게(darkening)** 하며, Command/Control 키를 누르면 **밝게(lightening)** 하는 대안 기능이 활성화됩니다.
  3. **결과**: 브러싱 후 조정 레이어가 생성되어 나중에 세부 조정이 가능합니다.
  4. **HSL 조정**: HSL(색조, 채도, 명도) 조정도 가능하며, 기본 기능은 채도를 높여 색상을 풍부하게 하는 것이고, Command/Control 키를 누르면 채도를 낮춥니다(desaturate).
2. **Burn Brush Tool (번 브러시 툴)**
  1. **기능**: 이미지 영역을 어둡게 하여 그림자를 강조합니다.
3. **Dodge Brush Tool (닷지 브러시 툴)**
  1. **기능**: 영역을 밝게 하여 하이라이트를 강조합니다.
  2. **주의**: 이 툴들 역시 **파괴적 편집** 방식으로, 나중에 취소할 수 있는 유연성이 떨어집니다.
4. **Sponge Brush Tool (스펀지 브러시 툴)**
  1. **기본 기능**: 기본적으로 이미지 영역의 **채도(saturate)**를 높입니다.
  2. **대안**: 채도를 낮추는(desaturate) 옵션도 제공됩니다.
  3. **주의**: 이 역시 파괴적으로 작동합니다.
5. **Filter Brush Tool (필터 브러시 툴)**
  1. **기능**: Adjustment Brush Tool과 매우 유사하며, 필터 효과를 선택하여 브러시로 칠할 수 있습니다.
  2. **작동 방식**: **Glitch(글리치)**와 같은 효과를 선택하고 브러시로 칠하면, 레이어 마스크가 추가되고 해당 효과가 적용됩니다.
  3. **조정**: 레이어 마스크를 통해 효과의 강도나 유형(예: Slice)을 변경할 수 있습니다.
  4. **접근**: 브러시를 사용하지 않고도 모든 라이브 필터에 접근할 수 있는 메뉴가 있습니다.

2.6. 블러 및 복제 툴

1. **Blur Brush Tool (블러 브러시 툴)**
  1. **작동 방식**: 블러를 적용하기 위해 사용하면, 어시스턴트가 나타나 레이어를 **래스터화**하도록 요청합니다.
  2. **옵션**: 상단에서 강도를 조절하고, 하단의 **Sharpen(선명하게)** 툴을 사용하여 디테일을 살릴 수 있습니다.
2. **Median Brush Tool (중앙값 브러시 툴)**
  1. **기능**: 스트로크 아래의 픽셀을 블렌딩하여 노이즈를 선택적으로 줄여줍니다.
  2. **평가**: 피부 보정 시 사용할 수 있으나, 파괴적으로 작동하여 디테일 손실 위험이 있습니다.
3. **Smudge Brush Tool (스머지 브러시 툴)**
  1. **활용**: 머리카락이나 털을 그릴 때 매우 유용할 수 있습니다.
  2. **예시**: 강도를 높여 민트 잎에 적용해 보았으나, 실제 유용한 상황에서 잘 활용될 수 있습니다.
4. **Clone Stamp Tool (복제 도장 툴)**
  1. **브러시 크기 조절**: 왼쪽/오른쪽 대괄호 키를 사용하여 브러시 크기를 조절할 수 있습니다.
  2. **샘플링**: Alt/Option 키를 누르고 클릭하여 영역을 샘플링합니다.
  3. **흐름(Flow)**: 복제 시 흐름(flow)과 불투명도(opacity)를 100%로 설정하는 것이 일반적입니다.
  4. **활용**: 특정 영역을 확장하거나, 원치 않는 요소를 복제하여 제거하는 데 유용합니다.
5. **Undo Brush Tool (실행 취소 브러시 툴)**
  1. **상태**: 사용자가 시도했으나 작동하지 않았으며, 히스토리 상태를 사용해야 하는 것으로 보입니다.

2.7. 리터칭 및 보정 툴

1. **Inpainting Brush Tool (인페인팅 브러시 툴)**
  1. **유사점**: Photoshop의 Remove Tool과 매우 유사합니다.
  2. **기능**: 제거하고 싶은 영역을 브러시로 칠하면(하이라이트 됨), 해당 영역이 제거됩니다.
  3. **결과**: 얼굴이나 팔 등을 제거하는 테스트 결과는 대체로 괜찮았으나, 하늘이나 덤불 같은 영역에서는 다른 얼굴이 나타나는 등 결과가 일정하지 않을 수 있습니다.
  4. **평가**: 빠르다는 장점이 있지만, 결과가 항상 완벽하지는 않습니다.
2. **Healing Brush Tool (복구 브러시 툴)**
  1. **유사점**: Photoshop의 Clone Brush Tool과 유사합니다.
  2. **기능**: 영역을 샘플링하여 이미지의 보기 흉한 부분을 수리하고 보정하는 데 사용됩니다.
3. **Patch Tool (패치 툴)**
  1. **기능**: 영역을 선택한 후, 다른 영역을 샘플링하여 그 영역으로 덮어씌울 수 있습니다.
  2. **활용**: 눈이나 팔 등을 복제하여 붙이는 등 리터칭에 유용합니다.
4. **Blemish Removal Tool (잡티 제거 툴)**
  1. **Red Eye Removal Tool**: 사용자가 사용해 본 적이 없어 설명은 생략합니다.
  2. **기능**: 제거하고 싶은 부위를 클릭 앤 드래그하여 선택하고, 원하는 영역을 스캔하며 자연스럽게 블렌딩되는 부분을 찾아 마우스를 놓으면 변경 사항이 확정됩니다.
  3. **평가**: 매우 유용하며, 흉터나 점을 제거하는 데 효과적입니다.

# 3. Vector Studio (재방문) 및 Pixel Studio 툴 재확인

3.1. Vector Studio 툴 재확인 (Pixel Studio 내에서)

1. **Pen Tool (펜 툴)**
  1. **기능**: 사용자 지정 모양을 그리거나, 채우기 색상을 줄 수 있습니다.
  2. **마스크/선택**: 아래 레이어를 마스크하거나, 생성된 모양을 사용하여 **선택 영역(selection)**을 만드는 데 주로 사용됩니다.
2. **Node Tool (노드 툴)**
  1. **기능**: 개별 앵커 포인트를 편집하는 데 사용되며, Illustrator의 Direct Selection Tool과 동등합니다.
3. **Shape Tools (도형 툴)**
  1. **캔버스 내 조정**: 도형을 생성한 후, **U**를 눌러 Shape Tool을 선택하면 캔버스 상에서 오렌지색 컨트롤 포인트를 드래그하여 설정을 직접 조정할 수 있습니다.
  2. **프리셋**: 톱니바퀴 모양 외에도 **방사능 기호(radioactive symbol)**와 같은 프리셋이 제공됩니다.

3.2. 텍스트 툴 (Pixel Studio 내에서)

1. **Artistic Text Tool (예술적 텍스트 툴)**
  1. **기능**: 클릭 후 타이핑하고 Move Tool로 크기를 조절하며, 제목이나 효과를 줄 텍스트에 적합합니다.
  2. **패널**: 상단에서 캐릭터 패널(Command/Control + T)과 단락 패널(Paragraph panel) 옵션을 확인할 수 있습니다.
  3. **유사점**: Illustrator의 Point Type과 유사합니다.
2. **Frame Text Tool (프레임 텍스트 툴)**
  1. **기능**: 상자를 드래그하여 텍스트를 입력하면 줄 바꿈이 적용됩니다.
  2. **텍스트 프레임 설정**: 텍스트 프레임 관련 설정 창이 따로 있으며, 텍스트 프레임의 파란색 상자(Text Flow)가 계속 보일 경우 **View > Show > Text Flow** 옵션을 꺼서 숨길 수 있습니다.

3.3. 기타 유틸리티 툴 (Pixel Studio 내에서)

1. **Color Picker Tool (색상 선택 툴)**
  1. **기능**: 이미지 영역 위에 호버링하여 색상 값을 확인합니다.
2. **Style Picker Tool (스타일 선택 툴)**
  1. **기능**: 선택된 레이어의 스타일 속성(색상, 획, 두께 등)을 복사하여 다른 객체에 적용합니다.
3. **Hand Tool (손 툴)**
  1. **단축키**: **H**이지만, 사용자는 더 빠른 접근을 위해 **Spacebar(스페이스바)** 사용을 선호합니다.
4. **Zoom Tool (확대/축소 툴)**
  1. **단축키**: **Z**를 사용하며, 확대/축소할 지점을 클릭합니다.
5. **Color Pickers (색상 선택기)**
  1. **기능**: 배경색과 전경색을 설정하며, **X**를 눌러 전환할 수 있습니다.
  2. **색상 선택기**: 더블 클릭하면 Photoshop과 유사한 색상 선택기가 나타납니다.

# 4. Layout Studio 툴 살펴보기 (Affinity 버전 InDesign)

Layout Studio는 InDesign과 유사한 레이아웃 및 출판 디자인 환경입니다.

4.1. 기본 이동 및 텍스트 툴

1. **Move Tool (이동 툴)**
  1. **기능**: 이미지 크기 조절, 회전, 이동에 사용되며, Shift를 누르면 15도 간격으로 스냅되며 회전합니다.
2. **Node Tool (노드 툴)**
  1. **기능**: 개별 포인트를 편집하는 데 유용하며, Illustrator의 Direct Selection Tool과 유사합니다. 사용자 지정 모양의 노드를 조정할 때 완벽합니다.
3. **Frame Text Tool (프레임 텍스트 툴)**
  1. **기능**: 텍스트 상자를 드래그하여 텍스트를 입력하거나 채움 텍스트를 추가할 수 있습니다.
  2. **패널 통합**: 이 스튜디오에서는 **Character(문자)** 및 **Paragraph(단락)** 옵션이 우측 패널 스위트(suite)에 통합되어 있습니다.
  3. **텍스트 래핑**: 텍스트 프레임을 조정하면 텍스트가 자동으로 여러 줄로 줄 바꿈됩니다.
4. **Artistic Text Tool (예술적 텍스트 툴)**
  1. **기능**: 클릭 후 타이핑하고 Move Tool로 크기를 조절하여 제목 텍스트 등에 사용합니다.
  2. **텍스트 추가 방식**: Layout Studio에서는 Frame Text Tool과 Artistic Text Tool, 두 가지 방식으로 텍스트를 추가할 수 있습니다.

4.2. 테이블 및 도형 툴

1. **Table Tool (테이블 툴)**
  1. **기능**: 드래그하여 테이블을 생성할 수 있습니다.
  2. **조정**: 핸들을 사용하여 행과 열의 개수를 조절할 수 있습니다.
  3. **셀 서식**: 셀을 선택하고 상단 옵션이나 **Table > Table Format** 메뉴를 통해 셀 색상 변경, 셀 높이/너비 조정, 패딩(여백) 추가 등을 할 수 있습니다.
2. **Pen Tool (펜 툴)**
  1. **기능**: 사용자 지정 모양을 만들거나, 아래 이미지/레이어를 마스크하는 데 사용됩니다.
  2. **선택**: 모양을 사용하여 선택 영역을 만들 수도 있습니다.
3. **Shape Tools (도형 툴)**
  1. **기능**: 별(star)과 같은 모양을 만들 수 있으며, 상단 옵션이나 **U**를 눌러 캔버스 상에서 직접 조정할 수 있습니다.
  2. **프리셋**: 꽃 모양 등 다양한 프리셋이 제공됩니다.
4. **Picture Frame Rectangle/Ellipse Tool (그림 액자 사각형/타원 툴)**
  1. **기능**: 문서에 **자리 표시자 콘텐츠(placeholder content)**를 추가하는 데 매우 유용하며, InDesign에서 와이어프레임 작업 시 유용합니다.

4.3. 배치 및 크롭 툴

1. **Place Tool (배치 툴)**
  1. **기능**: 이미지를 선택하여 배치합니다.
  2. **배치 방식**: 단일 클릭 시 기본 크기로 배치되거나, 클릭 앤 드래그로 크기를 지정하여 배치할 수 있습니다.
2. **Data Merge Layout Tool (데이터 병합 레이아웃 툴)**
  1. **기능**: 데이터 병합 시 새 페이지가 생성될 때 반복되는 그리드 레이아웃을 그립니다.
  2. **평가**: 사용자가 이 기능을 거의 사용하지 않아 자세한 설명은 추후 별도 영상에서 다룰 예정입니다.
3. **Vector Crop Tool (벡터 자르기 툴)**
  1. **기능**: 이미지 자르기에 더 중점을 둡니다.
  2. **작동 방식**: Pixel Studio의 Crop Tool과 달리, 이 툴은 **클리핑 마스크(clipping mask)**를 추가합니다.
  3. **복구**: 마스크를 삭제하면 원본 이미지가 복구됩니다.

4.4. 채우기 및 투명도 툴 (Layout Studio)

1. **Fill Tool (채우기 툴)**
  1. **문제 발생**: 이미지 위에 바로 적용 시 흑백으로 변하는 등 버그가 발생할 수 있습니다.
  2. **해결 방법**: 새 문서를 만들고, **Pixel > New Fill Layer**를 통해 채우기 레이어를 생성한 후, 해당 문서를 복사하여 붙여넣으면 정상 작동합니다.
  3. **채우기 유형**: 채우기 레이어에서 **Solid(단색)**, **Linear Gradient(선형 그라데이션)**, **Radial Gradient(방사형 그라데이션)**를 선택할 수 있습니다.
  4. **Bitmap**: **Bitmap** 옵션을 사용하여 스크린샷 등을 추가하여 패턴을 만드는 데 사용할 수 있으며, 매우 빠르고 반응성이 좋습니다.
2. **Transparency Tool (투명도 툴)**
  1. **기능**: 이미지에 적용 가능하며, 그라데이션 툴과 유사하게 이미지를 투명도로 페이드(fade) 처리합니다.
  2. **유형**: **Radial fade** 등 다양한 유형을 선택할 수 있으며, 반전(Reverse) 옵션으로 방향을 바꿀 수 있습니다.

4.5. 최종 유틸리티 툴 (Layout Studio)

1. **Color Picker Tool (색상 선택 툴)**
  1. **기능**: 색상을 선택하고 RGB 값을 확인하며, 레이아웃 모드에서는 이미지의 색상을 선택하면 해당 색상이 이미지의 색상으로 적용되는 것을 확인할 수 있습니다.
2. **Style Picker Tool (스타일 선택 툴)**
  1. **기능**: 한 이미지의 스타일 속성(색상, 획, 너비 등)을 복사하여 다른 레이어에 적용합니다.
3. **Hand Tool (손 툴)**
  1. **단축키**: **H**이지만, 스페이스바 사용을 선호합니다.
4. **Zoom Tool (확대/축소 툴)**
  1. **단축키**: **Z**를 사용하며, 특정 지점을 확대/축소합니다.
5. **Color Pickers (색상 선택기)**
  1. **기능**: 전경색과 획 색상을 설정하며, 색상 선택기를 열어 색상을 선택할 수 있습니다.
  2. **Layout 모드 적용**: Layout 모드에서는 선택한 색상이 **선택된 이미지**에 적용되는 것을 확인할 수 있습니다.

# 5. 마무리

1. **결론**: 이 영상은 Affinity 3의 모든 툴을 다루었으며, 분량이 많았음을 인정합니다.
2. **추가 정보**: 더 많은 Affinity 관련 영상은 링크된 다른 유튜브 채널에서 확인할 수 있습니다.
